# 2025年 AI Agent 记忆综述

> 知乎 | https://zhuanlan.zhihu.com/p/1985435669187825983

## 文章简介

这篇综述建立了 AI Agent 记忆的统一分类体系，系统梳理了超越 RAG 的下一代记忆架构。随着 AI Agent 从简单问答走向长期任务执行，记忆管理成为影响 Agent 能力上限的核心基础设施。

## 记忆分类体系

文章将 AI Agent 记忆分为四个层次：

### 1. 工作记忆（Working Memory）
- 对应：LLM 上下文窗口
- 特点：高速、高精度，但容量有限且不持久
- 优化方向：上下文压缩、重要性排序

### 2. 情节记忆（Episodic Memory）
- 对应：具体对话/任务的历史记录
- 特点：时序性，保留"何时发生什么"
- 实现：对话日志 + 时间戳索引

### 3. 语义记忆（Semantic Memory）
- 对应：提炼后的知识和事实
- 特点：去时序化，存储通用知识
- 实现：向量数据库 + 知识图谱

### 4. 程序性记忆（Procedural Memory）
- 对应：如何完成特定任务的技能
- 特点：隐式知识，体现在行为模式中
- 实现：Skills、工具使用模式

## 超越 RAG 的方向

- **主动记忆**：Agent 主动决定记什么，而非被动存储所有内容
- **记忆遗忘**：重要性衰减机制，避免低质量记忆积累
- **跨模态记忆**：文本、图像、代码统一的记忆表示
- **社会化记忆**：多 Agent 共享的集体记忆
