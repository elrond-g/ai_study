# graph-memory

> https://github.com/adoresever/graph-memory

## 项目简介

graph-memory 是一个基于图数据库的 AI 长期记忆系统，将 Graph RAG（图检索增强生成）应用于 AI Agent 的持久化记忆管理。相比传统向量 RAG，图记忆能够保存实体间的关系网络，支持更复杂的推理和知识关联。

## 核心功能

- **知识图谱构建**：自动从对话和文档中提取实体及关系，构建知识图谱
- **图 RAG 检索**：基于图结构的语义检索，支持多跳推理
- **长期记忆持久化**：记忆跨会话保存，实体关系持续积累
- **记忆遗忘机制**：模拟人类记忆的重要性衰减，避免信息爆炸

## 技术特点

- 基于 Neo4j 或轻量图数据库存储知识图谱
- 结合向量相似度和图遍历的混合检索
- 支持 LLM 自动抽取三元组（主体-关系-客体）

## Graph RAG vs 向量 RAG

| 维度 | Graph RAG | 向量 RAG |
|------|-----------|---------|
| 适合场景 | 关系推理、多跳查询 | 语义相似匹配 |
| 存储结构 | 图（节点+边） | 向量索引 |
| 推理能力 | 强（路径推理） | 弱 |
| 构建成本 | 较高 | 较低 |
