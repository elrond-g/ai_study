## 项目简介

claude-howto 是一份视觉化、示例驱动的 Claude Code 完整学习指南。从基础概念到高级 Agent 编程，配备生产就绪模板和 Mermaid 图表，15 分钟快速入门，11-13 小时完整学习路径。

## 10 个教程模块

| 模块 | 内容 |
|------|------|
| 斜杠命令 | 创建 /command-name 快捷方式，支持显式调用和自动触发 |
| 项目记忆 | 跨会话持久化上下文存储 |
| 技能（Skills） | 打包域名专业知识和工作流的文件系统组件 |
| 子代理 | 独立上下文窗口的专门化 AI 助手，任务委派 |
| MCP 集成 | 外部数据和工具集成的通用适配器 |
| 钩子（Hooks） | 会话中自动执行的脚本，自定义工作流 |

## 核心特色

- **15 分钟快速入门**：复制斜杠命令模板即获实际价值
- **生产级模板**：CLAUDE.md、钩子脚本、MCP 配置、子代理定义等即插即用
- **Mermaid 可视化**：每个特性的内部工作原理用图表直观展示
- **持续同步**：与每个 Claude Code 版本同步更新，v2.2.0（2026年3月）
- **兼容模型**：Claude Sonnet 4.6、Opus 4.6、Haiku 4.5

## 文件结构

按学习级别组织成 01-06 子目录，每个目录包含教程文档、示例代码和可视化图表。

## 为什么值得关注

填补了 Claude Code 学习资源的关键空白——从个人开发者到多 Agent 编排团队，从快速入门到生产部署的完整路径。模板优先的设计让开发者能直接复用最佳实践。
