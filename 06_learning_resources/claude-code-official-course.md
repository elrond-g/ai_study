# Claude Code in Action（官方实战课程）

> Anthropic 官方 | https://anthropic.skilljar.com/claude-code-in-action

## 课程简介

Anthropic 官方在 Skilljar 平台上发布的 Claude Code 实战课程，完成后可获得官方证书。系统讲解 Claude Code 如何通过工具系统读取文件、执行命令、修改代码，以及上下文管理、自定义工作流、Hooks 扩展和外部服务集成。

**适合人群**：希望用 AI 提速开发流程的工程师

**前置要求**：
- 基本的命令行使用经验
- Claude Code 访问权限 + API Key

---

## 学习目标

完成课程后，你将能够：

1. **核心工具使用** — 文件操作、命令执行、代码分析
2. **上下文管理** — 使用 `/init`、`CLAUDE.md`、`@` 提及控制上下文
3. **对话流程控制** — 掌握各类快捷键和命令
4. **深度分析模式** — 启用 Plan Mode 和 Thinking Mode 处理复杂任务
5. **自定义命令** — 创建命令自动化重复性开发工作流
6. **MCP 扩展** — 集成 MCP Server 添加浏览器自动化等能力
7. **GitHub 集成** — 配置自动 PR Review 和 Issue 处理
8. **Hooks 开发** — 编写 Hook 为 Claude Code 添加自定义行为
