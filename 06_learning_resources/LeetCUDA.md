# LeetCUDA

> ⭐ 10.6k | GPL-3.0 | https://github.com/xlite-dev/LeetCUDA

## 项目简介

面向 GPU 加速学习的 CUDA 教学资源库，提供 200+ CUDA Kernel 实现，从元素级操作到 FlashAttention-2 应有尽有。定位"从入门到精通"，附 100+ 技术博客深入讲解 GPU 优化与分布式计算。

## 核心内容

- **200+ CUDA Kernel**：涵盖元素运算到 FlashAttention-2
- **Tensor Core 实现**：WMMA / MMA / CuTe API 三种路径
- **HGEMM 达 cuBLAS 的 98%-100%**：工业级性能
- **FlashAttention-2 MMA 变体**：含内存优化细节
- **100+ 技术博客**：GPU 优化与分布式计算深度解析
- **Triton / CUTLASS 示例**：多个 Kernel 编写范式
- **PyTorch 绑定**：每个 Kernel 都有 Python 调用示例

## 技术覆盖

- CUDA C++ / PTX 汇编
- Triton DSL
- CUTLASS 模板库
- Python + PyTorch 绑定

## 学习路径

从 Element-wise 到 GEMM → Softmax → Attention → FlashAttention-2，循序渐进理解 GPU 编程核心范式，是从 CS336/cs149 理论到工业实现的绝佳桥梁。

## 使用场景

- 系统学习 CUDA 编程
- 面试准备（"LeetCUDA" 名字即致敬 LeetCode）
- AI 推理引擎优化研究
- 从 PyTorch 到底层 Kernel 的过渡学习
