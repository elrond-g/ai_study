## 项目简介

Claude Certified Architect — Foundations（CCA-F）认证的开源学习材料库。Anthropic 官方推出的 301 级别专业认证，面向解决方案架构师和 AI 工程师，考核生产级 Claude 应用架构能力。

## 考试结构

- 60 道单选题，120 分钟
- 要求至少 6 个月 Claude API + Claude Code 实战经验
- 费用：合作伙伴免费（前 5000 名），公开 $99/次

## 五大考试域

| 域 | 权重 | 内容 |
|------|------|------|
| Agent 架构与编排 | 27% | 多 Agent 系统、任务分解、中心辐射型模式 |
| 工具设计与 MCP 集成 | 18% | MCP 服务器设计、工具边界管理 |
| Claude Code 配置与工作流 | 20% | CLAUDE.md 配置、Agent 技能、MCP 集成 |
| Prompt 工程与结构化输出 | 20% | Few-shot、JSON Schema、验证循环 |
| 上下文管理与可靠性 | 15% | 长交互信息保留、错误处理、多 Agent 可靠性 |

## 考试场景（6 选 4）

客服 Agent、代码生成、多 Agent 研究系统、开发者生产力、CI/CD 集成、结构化数据提取

## 仓库内容

- 英文/俄文完整学习指南（guide_en.MD / guide_ru.MD）
- PDF 格式学习资料
- 涵盖工具设计、MCP、结构化输出、上下文管理等关键模式的实践练习
- 25 道练习题 + 12 周备考计划

## 为什么值得关注

Anthropic 首个专业级认证，系统覆盖从 Agent 架构、工具设计、代码工作流到成本优化的全方位知识。面向 Anthropic 合作伙伴网络，对 Claude 生态专家有职业价值。
