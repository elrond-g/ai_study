# awesome-ai-devtools

> GitHub | https://github.com/jamesmurdza/awesome-ai-devtools

## 项目简介

AI 驱动开发工具的精选列表，系统性收录和分类各类 AI 辅助开发工具，覆盖从 IDE、终端、Web 工具到自动化工作流的完整开发场景。3,700+ Stars。

## 分类覆盖

### 开发环境

- **AI 原生 IDE**：Cursor、Windsurf、Zed、Replit、Google Antigravity、Trae、PearAI、Crystal、Theia IDE 等
- **IDE 扩展**：GitHub Copilot、Cline、Continue、Tabnine、Supermaven、Amazon Q Developer、JetBrains AI、Refact AI 等

### 终端工具

- **终端 Agent**：Claude Code、Codex CLI、Gemini CLI 等
- **CLI 工具**：各类 AI 增强命令行工具

### Web 工具

- **App Builder**：v0、bolt.new、Lovable 等
- **UI 生成器**：AI 驱动的界面生成工具
- **编码 Agent**：Web 端编码代理
- **代码库智能**：代码理解和搜索工具
- **数据库 & SQL**：AI 辅助数据库工具

### 自动化工作流

- **PR & Code Review**：自动代码审查机器人
- **CI/CD & 测试**：自动化测试和持续集成

### Agent 基础设施

- 多 Agent 编排
- 沙盒与隔离
- 配置与上下文管理
- 用量分析与成本追踪

### 专门工具

- Git & Commit 助手
- 文档生成
- 代码搜索与理解
