# CS336：从零构建语言模型

> 斯坦福大学 | http://cs336.stanford.edu/

## 课程简介

CS336 是斯坦福大学开设的"Language Models from Scratch"课程，要求学生从零开始实现一个完整的大语言模型——包括 Transformer 架构、训练流程、优化器、分词器、评测等全链路。是目前最硬核的 LLM 工程教育课程之一。

## 课程内容

### 核心实现模块

1. **Tokenizer**：BPE 分词器从零实现
2. **Transformer 架构**：
   - Self-Attention 机制
   - Multi-Head Attention
   - Feed-Forward 层
   - Layer Normalization
   - Positional Encoding
3. **训练基础设施**：
   - 混合精度训练（FP16/BF16）
   - Gradient Checkpointing
   - 分布式训练基础
4. **优化器**：Adam、AdamW 手写实现
5. **评测**：Perplexity、下游任务评估

### 进阶主题

- RLHF 和对齐技术
- 高效推理（KV Cache、Flash Attention）
- 模型压缩与量化

## 为什么值得学

在 AI 工具唾手可得的时代，真正理解模型内部原理的工程师极为稀缺。CS336 的"从零实现"方法论——不允许调用现有框架的核心功能——强制建立从数学到代码的完整理解链路。
