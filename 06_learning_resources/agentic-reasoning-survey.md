# Agentic Reasoning for Large Language Models

> ArXiv | https://arxiv.org/abs/2601.12538 | https://arxiv.org/pdf/2601.12538

## 论文简介

Tianxin Wei 等 29 位作者合著的大型综述，系统梳理 LLM 作为自主 Agent 进行推理的理论框架。将 Agentic Reasoning 组织为三层能力栈：基础能力（规划/工具使用/搜索）、通过反馈和适应实现的自我进化、多 Agent 协作。

## 核心框架

- **In-context Reasoning**：测试时交互式推理，包括 CoT、ReAct、Reflection 等方法
- **Post-training Optimization**：通过 RL/SFT 优化 Agent 行为的训练时方法
- **Self-Evolution**：Agent 基于反馈和经验持续进化
- **Multi-Agent Collaboration**：多 Agent 分工协作完成复杂任务

## 关键贡献

- 提供 LLM 推理与真实世界行动连接的统一框架
- 覆盖科学研究、机器人、医疗、数学等跨学科应用
- 区分 in-context 与 post-training 两类推理范式
- 识别未来挑战：个性化、世界建模、部署治理

## 适用场景

- Agent 架构设计参考
- 系统性理解 LLM Reasoning 研究脉络
- 为多 Agent 系统和 Agent 自进化研究提供理论基础
