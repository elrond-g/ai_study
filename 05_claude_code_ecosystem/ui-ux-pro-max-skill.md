# ui-ux-pro-max-skill

为 AI 编码助手提供设计智能的 Skill 插件，支持跨多个平台构建专业级 UI/UX。

## 核心特性

- **AI 设计系统生成器**：秒级生成完整设计系统
- **丰富库存**：
  - 50+ 样式
  - 161 色板
  - 57 字体配对
  - 161 产品类型
  - 99 UX 指南
  - 10 个技术栈上的 25 种图表类型
- **10 技术栈支持**：React、Next.js、Vue、Svelte、SwiftUI、React Native、Flutter、Tailwind、shadcn/ui、HTML/CSS
- **多 Agent 平台**：Claude Code、Cursor、Windsurf、Antigravity、Codex CLI 等

## 安装

```bash
uipro init --ai claude
```
