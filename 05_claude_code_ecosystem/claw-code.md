# Claw Code

> ⭐ 174,000+ | Rust | https://github.com/ultraworkers/claw-code

## 项目简介

Claw Code 是基于 Rust 实现的 `claw` CLI Agent 框架，历史最快突破 100K Stars 的开源项目。基于 oh-my-codex 构建，提供强大的终端 Agent 编程能力。

## 核心功能

- **CLI Agent**：命令行原生的 AI 编程 Agent
- **Rust 高性能**：核心引擎 Rust 实现
- **生态丰富**：大量社区 Skills 和插件支持
- **多模型适配**：支持多种 LLM 后端

## 技术特点

- Rust 核心实现，极致性能
- 构建于 oh-my-codex 之上
- 社区驱动的快速迭代

## 使用场景

- 终端 AI 编程和项目开发
- Agent 编排与自动化任务执行
- 替代或补充 Claude Code 的开发场景
