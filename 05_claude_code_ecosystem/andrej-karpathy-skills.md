# andrej-karpathy-skills

> ⭐ 65.5k | MIT | https://github.com/forrestchang/andrej-karpathy-skills

## 项目简介

将 Andrej Karpathy 对 LLM 编码陷阱的观察（做错误假设、过度复杂化、乱改未理解的代码）提炼为 Claude Code 行为规范。以 Claude Code Plugin 或项目级 CLAUDE.md 形式集成，让 AI 输出更贴近资深工程师习惯。

## 四大核心原则

1. **Think Before Coding**（先思考后编码）：避免盲目假设，理清需求再动手
2. **Simplicity First**（简单优先）：反对过度工程，KISS 原则
3. **Surgical Changes**（外科式修改）：精准改动目标，不做无关改动
4. **Goal-Driven Execution**（目标驱动执行）：始终对齐用户真实目标

## 集成方式

- **Claude Code Plugin**：直接安装
- **项目级 CLAUDE.md**：复制到项目根目录
- **Cursor IDE**：支持 Cursor Rules 格式
- **自定义化**：可基于项目特性扩展规则
