# Claude-Code-Game-Studios

> ⭐ 3,649 | Shell | https://github.com/Donchitos/Claude-Code-Game-Studios

## 项目简介

把 Claude Code 变成一个完整的游戏开发工作室。内置 48 个 AI Agent 和 36 个工作流 Skills，完整模仿真实游戏工作室的层级结构，覆盖从策划、美术、编程到测试的全链路。

## 核心功能

- **48 个专业 Agent**：制作人、游戏设计师、程序员、美术、音效、QA 等各角色分工明确
- **36 个工作流 Skills**：针对游戏开发各阶段（关卡设计、物理系统、AI 行为等）的专项 Skill
- **工作室层级协调**：主管 Agent 拆解任务并分配给专业子 Agent，模拟真实工作室流程
- **多引擎支持**：覆盖 Unity、Unreal Engine、Godot 的工作流

## 适用引擎

- Unity（C#）
- Unreal Engine（C++ / Blueprint）
- Godot（GDScript）

## 使用场景

- 独立游戏开发者用 AI 替代完整团队
- 快速原型验证游戏设计方案
- 学习游戏开发工作室标准化流程
