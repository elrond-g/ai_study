# ljg-skills

lijigang 的 Claude Code Skills 合集，包含多个专业工具用于知识探索、学术分析、创意写作和深度对话。

## Skills 列表

- **ljg-travel**：生成深度文化研究文档（org-mode 格式）+ 便携 PNG 卡片，覆盖历史、博物馆、建筑、考古
- **ljg-roundtable**：结构化辩论框架，主持人引导历史/当代人物多轮讨论，含 ASCII 思维图和知识网络
- **ljg-word**：英语单词深度语义分析和洞察生成
- **ljg-writes**：写作引擎，在写作过程中发展想法
- **ljg-skill-map**：可视化展示所有已安装 Skills 概览
- **ljg-explain-concept**：8 维度概念解剖器，将任意概念拆解为关键洞察
- **ljg-skill-xray-paper**：学术论文分析 Skill，支持 arxiv ID、URL、PDF，提取核心概念

## 特性

- 支持工作流串联（workflow chaining），组合多个 Skill 完成复杂任务
