# Waza

> ⭐ 3.6k | MIT | https://github.com/tw93/Waza

## 项目简介

tw93 出品的 Claude Code Skills 集合，将工程师日常的工作习惯（系统思考、设计评审、调试、代码检查等）转化为 AI 可直接执行的技能。定位是"质而不是量"——8 个聚焦的 Skill，经过 500+ 小时实战打磨。

## 8 个核心命令

| 命令 | 用途 |
|------|------|
| `/think` | 系统化思考框架 |
| `/design` | 设计方案生成与评审 |
| `/check` | 代码检查 |
| `/hunt` | Bug 定位与调试 |
| `/write` | 写作辅助 |
| `/learn` | 学习新技术 |
| `/read` | 阅读理解 |
| `/health` | 项目健康检查 |

## 附加功能

- **Statusline**：显示上下文使用量和 API 配额
- **English Coaching Rule**：提示词精炼规则
- 兼容 Claude Code 与 Codex 双环境

## 安装方式

```bash
npx skills add tw93/Waza -a claude-code -g -y
```
