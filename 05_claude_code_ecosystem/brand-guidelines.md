## 项目简介

Anthropic 官方的品牌规范 Skill，能够自动将品牌指南应用到 Claude 的输出中，确保生成内容始终符合企业品牌标准。

## 核心功能

- **品牌规范解析**：读取并理解企业品牌手册中的视觉和文字规范
- **自动风格应用**：在生成内容时自动遵循品牌色彩、字体、语气等规范
- **一致性检查**：验证输出内容是否符合品牌指南要求

## 技术特点

- 支持导入自定义品牌规范文档
- 涵盖视觉设计和文案语气双重规范
- 与其他设计类 Skill 协同工作，形成完整的品牌输出链路

## 使用场景

- 企业团队统一 AI 生成内容的品牌调性
- 确保营销物料和产品界面的视觉一致性
- 新成员快速了解并遵循品牌设计规范
