# ClawTeam

> https://github.com/HKUDS/ClawTeam

## 项目简介

ClawTeam 是香港大学数据科学实验室开源的 Claw（Claude Agent）自组队系统，让多个 Claw Agent 根据任务需求自动组建专业团队、分配角色并协同完成复杂任务，无需人工指定团队结构。

## 核心机制

**自组队流程：**
1. 接收复杂任务描述
2. 系统分析任务需求，确定所需专业技能
3. 自动实例化相应角色的 Claw Agent（PM、工程师、设计师等）
4. 动态分配子任务，监控进度
5. 汇总各 Agent 产出，生成最终结果

## 核心功能

- **自动角色推断**：根据任务自动决定需要哪些专业角色
- **动态团队组建**：运行时按需创建 Agent 实例
- **任务协调**：Agent 间的依赖管理和进度同步
- **结果整合**：将多个 Agent 的产出合并为一致的最终交付物

## 技术特点

- 与 CLI-Anything（同一实验室）的工具生态兼容
- 支持并行 Agent 执行，提升效率
- 内置冲突解决机制

## 使用场景

- 需要多专业角色协同的复杂软件项目
- 研究多 Agent 系统的自组织能力
- 构建无需人工设计团队结构的 AI 项目管理系统
