## 项目简介

Anthropic 官方出品的前端设计 Skill，帮助 Claude 生成符合现代设计规范的前端界面，告别 AI 生成页面的"塑料感"。已有超过 27 万次安装，是最受欢迎的官方 Skill 之一。

## 核心功能

- **设计系统集成**：内置完整的设计系统规范，确保生成的 UI 具有一致的视觉语言
- **组件化输出**：生成可复用的前端组件，支持主流框架的代码输出
- **响应式布局**：自动处理不同屏幕尺寸的适配，生成真正可用的响应式页面

## 技术特点

- 基于 Anthropic 官方设计规范，输出质量有保障
- 支持 Tailwind CSS 等主流样式方案
- 生成代码遵循无障碍访问（a11y）最佳实践

## 使用场景

- 快速搭建产品原型和落地页
- 将设计稿转化为前端代码
- 统一团队 UI 输出风格，减少手动调整
