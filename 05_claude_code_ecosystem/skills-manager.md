# skills-manager

> ⭐ 370 | Rust | https://github.com/xingkongliang/skills-manager

## 项目简介

轻量级桌面应用，专为管理和同步 AI Agent Skills 而设计，支持 Cursor、Claude Code、Codex、GitHub Copilot 等 15+ 主流 AI 编程工具，解决 Skills 分散管理、手动安装繁琐的问题。

## 核心功能

- **统一管理**：在一个界面管理所有 AI 工具的 Skills，不再需要逐个工具单独配置
- **跨工具同步**：一套 Skills 同步到多个工具，保持一致的 AI 辅助体验
- **Skills 组织**：分类、标签、搜索，快速找到所需 Skill
- **一键安装/卸载**：GUI 操作，无需手动修改配置文件

## 支持的工具

Cursor、Claude Code、Codex、GitHub Copilot、Windsurf、Continue、Aider 等 15+ 工具

## 技术特点

- Rust 实现，轻量高效，启动快
- 桌面 GUI 应用，跨平台（macOS/Windows/Linux）
- 本地存储，Skills 数据不上传云端

## 使用场景

- 同时使用多个 AI 编程工具的开发者
- 团队共享 Skills 配置，保持一致的 AI 辅助规范
- 频繁尝试和切换不同 Skills 的用户
