## 项目简介

Anthropic 官方 Claude Code Skill，用自然语言操作 Excel 文件，支持公式编写、数据分析和图表生成。

## 核心功能

- **自然语言操作表格**：用中文或英文描述需求即可创建、编辑和格式化 Excel 工作表
- **公式与函数生成**：自动生成 VLOOKUP、SUMIFS 等复杂公式，支持跨表引用
- **数据分析与可视化**：内置数据透视、排序筛选功能，可直接生成图表

## 技术特点

- 基于 openpyxl 库实现对 .xlsx 格式的完整读写支持
- 保留原有样式、条件格式和数据验证规则
- 支持大规模数据集处理，适用于万行级别表格

## 使用场景

- 自动化生成财务报表和数据汇总表
- 对导入的 CSV/Excel 数据进行清洗和分析
- 批量为多个工作表添加公式和图表
