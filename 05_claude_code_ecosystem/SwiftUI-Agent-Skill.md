# SwiftUI-Agent-Skill

> https://github.com/twostraws/SwiftUI-Agent-Skill

## 项目简介

SwiftUI-Agent-Skill 是 Paul Hudson（twostraws，Hacking with Swift 创始人）为 Claude Code 开发的 iOS/macOS SwiftUI 专项 Skill。将 Paul 多年 Swift 教学和工程经验编码为 AI 可执行的苹果平台开发规范。

## 核心功能

- **SwiftUI 最佳实践**：视图组织、状态管理、生命周期规范
- **Swift 并发**：async/await、Actor 模型的正确使用
- **苹果平台规范**：HIG（人机界面指南）的代码层面落实
- **常见错误预防**：SwiftUI 的 identity、lifetime 等易踩坑点
- **性能优化**：避免不必要的视图刷新

## 为什么值得关注

Paul Hudson 是 Swift 社区最知名的教育者之一，其 Hacking with Swift 系列覆盖了大量苹果开发者。这套 Skill 浓缩了他多年的教学精华，对于用 Claude Code 做 iOS/macOS 开发的开发者价值极高。

## 使用场景

- iOS/macOS 开发者用 Claude Code 加速 SwiftUI 开发
- 快速验证代码是否符合苹果平台最佳实践
- 新手学习 SwiftUI 时获得规范化的 AI 指导
