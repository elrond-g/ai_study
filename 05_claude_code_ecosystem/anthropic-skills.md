## 项目简介

Anthropic 官方 Skills 仓库，包含所有官方出品的 Skill 源码、规范文档和开发指南。是了解 Skills 标准和学习 Skill 开发的权威参考。

## 核心功能

- **官方 Skill 源码**：提供 Frontend Design、Canvas Design 等所有官方 Skill 的完整源码
- **开发规范文档**：定义 Skill 的标准格式、目录结构和最佳实践
- **示例与模板**：为社区开发者提供 Skill 开发的起步模板和参考实现

## 技术特点

- 所有 Skill 遵循统一的 CLAUDE.md 规范格式
- 代码质量经 Anthropic 团队审核，可作为开发标杆
- 持续迭代更新，反映 Claude Skills 平台的最新能力

## 使用场景

- 学习 Skill 的标准开发流程和规范
- 阅读官方 Skill 源码，理解高质量 Skill 的设计思路
- Fork 并修改官方 Skill 以适配自身需求
