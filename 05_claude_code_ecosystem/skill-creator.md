# Skill Creator

> ⭐ N/A | Markdown (Skill) | https://github.com/anthropics/skills/tree/main/skills/skill-creator

## 项目简介

Anthropic 官方元技能（Meta Skill），帮助用户在 5 分钟内快速创建自定义 SKILL.md 文件，降低 Skill 开发门槛。

## 核心功能

- **交互式 Skill 生成**：通过对话式引导收集需求，自动生成符合规范的 SKILL.md
- **模板与最佳实践**：内置多种 Skill 模板，遵循 Anthropic 官方结构和命名规范
- **即时验证与测试**：生成后可立即在 Claude Code 中加载测试，快速迭代

## 技术特点

- 元编程设计——用 Skill 来创建 Skill，体现 Claude Code 的自举能力
- 自动生成标准化的指令段、工具调用模式和错误处理逻辑
- 支持从现有脚本或工作流反向生成对应的 Skill 配置

## 使用场景

- 团队内快速将常用工作流封装为可复用的 Claude Code Skill
- 初学者学习 Skill 编写规范和最佳实践
- 将已有 Shell 脚本或 Python 工具包装为 Claude Code 插件
