# ppt-agent-workflow-san

> GitHub | https://github.com/mucsbr/ppt-agent-workflow-san

## 项目简介

渐进交互式 PPT 生成 Skill，通过与 Agent 简单对话即可生成专业级演示文稿。采用渐进式交互方式，每次对话逐步完善 PPT 内容和设计。

## 效果示例

项目展示了以"汽水音乐"为调研主题，经过与 Agent 几轮对话后生成的 PPT 效果：

- Cover（封面）
- Core Conclusion（核心结论）
- Positioning（产品定位）
- Users Scenarios（用户场景）
- Growth Flywheel（增长飞轮）
- Competition（竞争分析）
- Risks（风险分析）
- Conclusion（总结）
