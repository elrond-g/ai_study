## 项目简介

Anthropic 官方 Claude Code Skill，实现人机协作写作模式——人写一段、Claude 接一段，交替完成长文档创作。

## 核心功能

- **交替写作模式**：用户撰写段落后 Claude 自动续写，保持风格和逻辑连贯
- **风格适配**：自动分析已有文本的语气、用词和结构，确保续写内容无缝衔接
- **版本与差异追踪**：支持查看每轮协作的修改内容，方便回溯和调整

## 技术特点

- 基于上下文窗口动态管理已写内容，确保长文不丢失前文信息
- 支持多种文体：技术文档、博客文章、小说创作等
- 可与 DOCX Skill 联合使用，直接输出为 Word 格式

## 使用场景

- 与 Claude 合作撰写技术博客或研究报告
- 创意写作中用 AI 突破写作瓶颈
- 团队文档协作中由 AI 补充草稿内容
