# Qiaomu Design Advisor

> ⭐ 228 | MIT | https://github.com/joeseesun/qiaomu-design-advisor

## 项目简介

面向 Claude Code 的 AI 设计顾问 Skill，融合乔布斯的产品直觉与 Dieter Rams 的功能极简主义，帮助用户从零打造像素级完美的设计稿，消除设计决策中的主观猜测。

## 核心特点

- **三阶段工作流**：诊断（Diagnosis）→ 方案（Proposals）→ 执行（Execution）
- **58 套真实设计系统**：提取自 Stripe、Vercel、Linear 等知名公司的可直接参考的设计模式库
- **10 种设计原型推荐**：通过诊断问题自动匹配最合适的风格方向
- **对话式交互**：以自然语言对话引导用户完成设计决策

## 使用方式

- 作为 Claude Code Skill 安装
- 需要 Node.js 18+
- 所有设计系统以 Markdown 文件形式存储，可读性强

## 使用场景

- 非设计师开发者需要生成专业级 UI 设计稿
- 产品早期需要快速对比不同设计风格方向
- 项目迭代中保持视觉一致性
