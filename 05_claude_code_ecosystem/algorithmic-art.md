## 项目简介

Anthropic 官方的算法艺术 Skill，基于 p5.js 生成分形图案、几何构图等程序化艺术作品。让 Claude 成为你的算法艺术创作助手。

## 核心功能

- **分形图案生成**：创建曼德博集合、Julia 集等经典分形图案
- **几何构图设计**：利用数学公式生成对称、递归的几何艺术
- **交互式动画**：生成可交互的 p5.js 动画作品，支持实时参数调节

## 技术特点

- 基于 p5.js 创意编程框架，输出标准 Web 可运行代码
- 支持噪声函数、粒子系统等高级图形技术
- 生成的代码结构清晰，易于二次修改和学习

## 使用场景

- 创作生成式艺术作品用于展览或 NFT
- 学习算法艺术和创意编程的入门工具
- 为网页或应用生成独特的动态背景和装饰元素
