# clui-cc

> https://github.com/lcoutodemos/clui-cc

## 项目简介

clui-cc 是 Claude Code 的 UI 封装层，为 Claude Code 提供更友好的可视化交互界面。在保留 Claude Code 全部底层能力的同时，提供更直观的界面组件和交互模式。

## 核心功能

- **增强 UI 组件**：比默认 Claude Code 界面更丰富的可视化元素
- **Claude Code 完整集成**：底层仍调用 Claude Code，无功能损失
- **自定义主题**：可调整界面风格和布局
- **便捷操作**：常用命令的快捷入口

## 技术特点

- 基于 Claude Code CLI 构建，非独立实现
- 提供额外的 UI 层而不替换底层
- 轻量设计，保持 Claude Code 的响应速度

## 使用场景

- 希望在保留 Claude Code 能力的同时改善视觉体验
- 演示和分享 Claude Code 工作流时需要更好看的界面
- 探索 Claude Code 的 UI 扩展可能性
