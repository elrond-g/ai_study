## 项目简介

Anthropic 官方的 Web 构件生成 Skill，通过自然语言描述即可生成计算器、Dashboard、表单等实用 Web 小工具。将想法快速变成可交互的网页应用。

## 核心功能

- **自然语言构建**：用中文或英文描述需求，直接生成可运行的 Web 应用
- **Dashboard 生成**：快速创建数据看板，支持图表、指标卡片等常见组件
- **实用工具输出**：生成计算器、转换器、表单等日常工具类页面

## 技术特点

- 输出单文件 HTML，无需构建工具即可直接运行
- 内置 Chart.js 等可视化库的集成能力
- 生成代码包含完整的交互逻辑和样式

## 使用场景

- 快速制作内部数据 Dashboard 原型
- 生成各类在线计算器和转换工具
- 创建表单收集页和交互式演示页面
