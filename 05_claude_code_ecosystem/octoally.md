# octoally

> ⭐ 60 | TypeScript | https://github.com/ai-genius-automations/octoally

## 项目简介

AI 编码会话编排仪表板，提供 Web UI 来启动、监控和管理 Claude Code 会话，让开发者可以在浏览器中统一管理多个并行的 Claude Code 工作流。

## 核心功能

- **会话启动**：从 Web UI 一键创建和启动 Claude Code 会话
- **实时监控**：在仪表板中查看所有活跃会话的状态、进度和输出
- **多会话管理**：同时管理多个并行运行的 Claude Code 实例
- **历史记录**：查看历史会话的执行结果和日志

## 技术特点

- TypeScript 实现，Web 界面，无需安装桌面客户端
- 与 Claude Code CLI 深度集成
- 适合需要并行运行多个 AI 编码任务的场景

## 使用场景

- 同时运行多个代码生成/修改任务，提高并行效率
- 团队共享 Claude Code 会话，协同工作
- 需要可视化界面管理 AI 编程工作流的场景
