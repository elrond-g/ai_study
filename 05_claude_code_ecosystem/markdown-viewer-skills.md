# Markdown Viewer Skills

> ⭐ 1.9k | MIT | https://github.com/markdown-viewer/skills

## 项目简介

为 AI 编码 Agent 设计的 Skills 合集，让 AI 可以直接在 Markdown 中生成专业级图表和可视化。14 个 Skill 覆盖 5 种渲染引擎，输出即嵌入 Markdown 中无缝显示。

## 14 个 Skill 覆盖范围

- **UML 图**：类图、时序图、状态图
- **云架构图**：AWS/Azure/GCP 架构可视化
- **网络拓扑**：网络和安全架构图
- **BPMN 工作流**：业务流程建模
- **数据分析**：Vega-Lite 统计图表
- **IoT 图**：物联网架构可视化
- **信息图**：70+ 模板的 Infographic
- **编辑级信息卡**：出版级 Info Card

## 渲染引擎支持

- PlantUML：9,500+ 图标 Stencil 库
- Vega / Vega-Lite：数据可视化
- Canvas JSON：自定义画布格式
- HTML/CSS：嵌入式可视化
- Markdown 原生渲染

## 使用场景

- 技术文档自动配图（架构图、流程图）
- AI 生成报告时嵌入可视化数据
- 代码库文档的自动图表生成
