# Research Skills

> ⭐ 429 | TypeScript | https://github.com/luwill/research-skills

## 项目简介

Research Skills 是一个 Claude Code 学术科研 Skill 合集，为科研工作者提供多个专业化的研究工作流，包括医学影像综述撰写、论文幻灯片生成和博士研究计划书生成。

## 核心功能

- **medical-imaging-review**：医学影像文献综述自动撰写 Skill
- **paper-slide-deck**：论文转演示幻灯片 Skill
- **research-proposal**：博士研究计划书生成 Skill
- **Zotero 集成**：支持文献管理工具 Zotero 的数据导入
- **MCP 支持**：集成 ArXiv 和 PubMed MCP 服务

## 技术特点

- TypeScript 实现
- 遵循 Claude Code Skill 规范
- 支持学术数据源的结构化接入

## 使用场景

- 学术论文综述写作辅助
- 学术演讲 PPT 快速生成
- 博士/硕士研究计划书草拟
