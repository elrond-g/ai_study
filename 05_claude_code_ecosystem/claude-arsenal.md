# claude-arsenal

> ⭐ 12 | Python | https://github.com/majiayu000/claude-arsenal

## 项目简介

面向专业软件开发的 Claude Code Skills 综合库，包含 39+ 久经验证的 Skills 和 9 个专业 Agent，同时提供覆盖产品完整生命周期的 Skills 路线图。

## 核心内容

### 39+ Skills 分类

- **语言类**：Python、TypeScript、Go、Rust 等各语言开发规范
- **框架类**：React、FastAPI、Django 等主流框架最佳实践
- **工程类**：API 设计、测试策略、代码审查、重构指南
- **运维类**：DevOps 流程、可观测性、部署规范

### 9 个专业 Agent

- product-discovery（产品发现）
- prd-master（PRD 撰写）
- product-ux-expert（UX 设计）
- comprehensive-tester（测试）
- devops-excellence（DevOps）
- observability-master（可观测性）等

## 产品全生命周期路线图

```
发现 → 定义 → 设计 → 开发 → 测试 → 发布 → 运营
```

每个阶段对应专属 Skills 和 Agent，覆盖产品从 0 到 1 的完整流程。

## 使用场景

- 独立开发者需要覆盖产品开发全链路的 AI 辅助
- 团队建立统一 AI 编程规范
- PM/开发者协作的跨角色 AI 工作流
