# Memento-Skills

"让 Agent 设计 Agent" — 自主学习的 Agent 框架，将技能作为一等公民，通过持续学习循环自动进化和优化。

## 核心特性

- **技能库系统**：可检索、可执行、可持久化、可演进的技能管理
- **技能路由器**：根据任务自动检索已有技能或生成新技能
- **反思循环**：成功则提高效用分数，失败则优化和再生成技能
- **完整栈**：自有编排、技能路由、执行、反思、存储
- **多接口**：CLI、GUI、IM 集成
- **本地执行**：持久化状态，数据不离开本机

## 工作流程

1. 接收任务 → 2. 技能路由器检索匹配 → 3. 无匹配则自动生成新技能 → 4. 执行 → 5. 反思评估 → 6. 更新技能效用分数 → 循环
