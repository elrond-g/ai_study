# sisyphus-design

为 AI 编码代理设计的迭代式设计文档生成 Skill，通过循环流程和轮换评审员机制实现高质量设计输出。

## 核心特性

- **迭代流程**：Research（研究）→ Diverge（发散）→ Decide（决策）→ Draft（草稿）→ Refine（细化）
- **轮换盲审机制**：每轮评审员都是全新的，对设计一无所知，确保严格审视
- **设计输出**：生成到项目 `design/` 目录
- **多平台支持**：Claude Code 和 Codex
- **命令**：`/sisyphus-design <requirement>`
