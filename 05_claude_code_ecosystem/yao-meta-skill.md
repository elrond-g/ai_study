# yao-meta-skill

> ⭐ 275 | Python | https://github.com/yaojingang/yao-meta-skill

## 项目简介

YAO（Yielding AI Outcomes）元技能系统，将工作流、提示词和文档转化为可复用的 Agent 技能包，内置评估、治理、跨平台导出流程。

## 核心功能

- **工作流转技能**：将现有工作流自动转化为 Agent Skill
- **提示词转技能**：从提示词模板生成标准化技能包
- **评估机制**：内置技能质量评估流程
- **跨平台导出**：支持导出到多种 Agent 平台

## 技术特点

- Python 实现
- 元技能（Meta-Skill）架构
- 标准化技能包格式

## 使用场景

- 将个人经验快速转化为 Agent Skill
- 技能包的标准化生产和分发
- 跨平台 Agent 技能迁移
