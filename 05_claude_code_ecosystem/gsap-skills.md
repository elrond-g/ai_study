# gsap-skills

> ⭐ 1,561 | https://github.com/greensock/gsap-skills

## 项目简介

GreenSock（GSAP）官方为 AI 编程代理发布的 Skills 集合，教导 AI 如何正确使用 GSAP 动画库，包含最佳实践、常见动画模式和插件用法。由 GSAP 原厂团队维护，是最权威的 GSAP AI 辅助开发资源。

## 核心功能

- **GSAP 核心用法**：`gsap.to()`、`gsap.from()`、`gsap.timeline()` 等基础 API 的正确用法
- **插件覆盖**：ScrollTrigger、Draggable、MorphSVG、SplitText 等热门插件指南
- **常见动画模式**：滚动动画、视差效果、页面过渡等典型场景的实现方式
- **避坑指南**：AI 常见的 GSAP 误用和错误模式

## 为什么需要这个 Skill

AI 对 GSAP 的训练数据较旧，常会生成过时或错误的代码（如使用废弃 API、错误的 ScrollTrigger 配置）。这个 Skill 向 AI 注入最新、最准确的 GSAP 知识。

## 使用场景

- 前端开发者用 Claude Code / Cursor 开发 GSAP 动画
- 需要 AI 生成高质量动画代码而非凭训练数据猜测
- 学习 GSAP 最佳实践
