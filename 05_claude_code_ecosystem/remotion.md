# Remotion Best Practices

> ⭐ N/A | TypeScript | https://github.com/remotion-dev/remotion

## 项目简介

Remotion 是基于 React 的程序化视频生成框架，周安装量达 11.7 万，可结合 AI 实现自动化视频制作。

## 核心功能

- **React 驱动视频**：用 React 组件定义视频的每一帧，前端开发者可直接上手
- **程序化渲染**：支持参数化批量渲染视频，可集成 CI/CD 实现自动化生产
- **AI 视频生成**：结合 Claude 等 AI 模型自动生成视频脚本和动画参数

## 技术特点

- 基于 Chromium 逐帧渲染，输出质量等同浏览器渲染精度
- 支持 SSR 服务端渲染和 Lambda 云端渲染，适配不同规模需求
- TypeScript 原生支持，类型安全，生态丰富（音频、字幕、动画插件）

## 使用场景

- 批量生成个性化营销视频或产品演示
- 数据可视化动画自动生成
- 结合 AI 将文本内容自动转化为短视频
