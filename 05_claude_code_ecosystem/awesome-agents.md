## 项目简介

AI Agent 开源项目精选列表，覆盖框架、工具、平台等 Agent 生态全景。是了解和跟踪 AI Agent 领域发展的综合导航资源。

## 核心功能

- **生态全景索引**：涵盖 Agent 框架、开发工具、部署平台等完整技术栈
- **分类导航**：按功能类型和应用场景组织项目，便于快速定位
- **趋势追踪**：持续收录新兴 Agent 项目，反映行业最新动态

## 技术特点

- 覆盖 LangChain、CrewAI、AutoGPT 等主流 Agent 框架
- 包含从底层模型到上层应用的全链路工具
- 社区活跃，更新频率高，信息时效性强

## 使用场景

- 调研 AI Agent 技术选型时的参考指南
- 了解 Agent 生态的整体格局和发展方向
- 发现可集成到自身项目中的 Agent 工具和框架
