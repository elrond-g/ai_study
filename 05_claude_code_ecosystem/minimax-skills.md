# MiniMax-AI/skills

> ⭐ 3,660 | https://github.com/MiniMax-AI/skills

## 项目简介

MiniMax 官方发布的 AI 编程 Skills 集合，为 AI 编码代理提供前端、全栈、Android、iOS、Shader 等多个技术方向的结构化专业指导，可直接挂载到 Claude Code、Cursor、Codex 等工具。

## 覆盖技术方向

| Skill | 说明 |
|------|------|
| 前端开发 | React/Vue 最佳实践、性能优化、组件设计 |
| 全栈开发 | API 设计、数据库交互、部署规范 |
| Android | Kotlin/Jetpack Compose 开发规范 |
| iOS | Swift/SwiftUI 开发规范 |
| Shader | GLSL/HLSL 着色器编写指南 |

## 核心功能

- **生产级指导**：强调可上线质量，不仅是能跑的代码
- **官方背书**：由 MiniMax 工程团队维护，与内部开发规范对齐
- **标准 SKILL.md 格式**：兼容所有主流 AI 编程工具

## 使用场景

- 需要 AI 生成符合生产标准的多平台代码
- 团队统一 AI 编码规范
- 跨平台开发时快速切换不同技术栈的 AI 指导
