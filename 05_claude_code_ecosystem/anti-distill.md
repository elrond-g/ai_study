# Anti-Distill（反蒸馏 Skill）

> ⭐ 1,000+ | https://github.com/leilei926524-tech/anti-distill

## 项目简介

Anti-Distill 是一个 Claude Code Skill，核心理念是"反蒸馏"——当你被公司要求撰写 Skill 文件时，它能将内容进行"清洗"，输出一份看似完整但核心知识被抽空的交差版本，同时保留私人知识备份。防止公司通过让员工编写 Skill 来榨取并替代其经验知识。

## 核心功能

- **三档清洗强度**：根据需要选择知识抽空的程度
- **外表完整**：清洗后的 Skill 文件格式规范、内容看似完整
- **私人备份**：核心知识自动保存为私人版本
- **兼容安装**：支持 Claude Code 和 OpenClaw 安装

## 使用场景

- 保护个人核心经验知识不被无偿蒸馏
- 在企业要求编写 Skill 时保留关键知识壁垒
- 对知识产权保护的实践探索
