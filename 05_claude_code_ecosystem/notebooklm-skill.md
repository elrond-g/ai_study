## 项目简介

集成 Google NotebookLM 功能的 Claude Skill，支持文档摘要、思维导图生成和闪卡制作。将 NotebookLM 的知识管理能力引入 Claude 工作流。

## 核心功能

- **智能摘要**：对长文档进行结构化摘要，提取核心观点和关键信息
- **思维导图生成**：自动将文档内容组织为思维导图结构，可视化知识脉络
- **闪卡制作**：从文档中提取知识点，生成问答式闪卡用于记忆和复习

## 技术特点

- 与 Google NotebookLM 生态打通，支持导入多种文档格式
- 输出结构化的 Markdown 格式，便于后续编辑和分享
- 支持多文档交叉分析和知识关联

## 使用场景

- 学术论文或技术文档的快速阅读和摘要
- 课程学习时制作复习闪卡和知识框架
- 团队知识库的内容整理和结构化
