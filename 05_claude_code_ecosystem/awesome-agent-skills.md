# awesome-agent-skills

> https://github.com/libukai/awesome-agent-skills

## 项目简介

awesome-agent-skills 是一份精心整理的 Claude Code / AI Agent Skills 聚合列表，收录来自社区的优质 Skill 资源，方便开发者发现和评估各类 Skills。是探索 Claude Code Skills 生态的重要入口之一。

## 收录范围

- **开发类 Skills**：前端、后端、全栈、移动端、DevOps
- **设计类 Skills**：UI/UX、设计系统、视觉规范
- **研究类 Skills**：学术写作、文献综述、数据分析
- **生产力类**：写作助手、项目管理、文档生成
- **专业领域**：法律、医疗、金融等垂直行业

## 如何使用

1. 在列表中找到适合场景的 Skill
2. 按 Skill 的安装说明添加到 Claude Code
3. 使用对应的 `/命令` 激活 Skill

## 与其他聚合对比

| 聚合项目 | 特点 |
|---------|------|
| awesome-agent-skills（本项目） | 精选，有质量筛选 |
| agent-skills-hub | 数量多，社区驱动 |
| readyclaw.org | 数万个 Skills，搜索友好 |
