# awesome-design-md

> ⭐ 61.3k | MIT | https://github.com/VoltAgent/awesome-design-md

## 项目简介

精选 DESIGN.md 文件集合——将设计系统以纯文本描述文件形式沉淀，AI Agent 读取后可生成符合特定品牌风格的 UI。69+ 个 DESIGN.md 覆盖 Claude、Figma、Stripe、Apple 等知名产品的设计系统。

## 核心价值

- **69+ 开箱即用的 DESIGN.md**：从知名网站提取的设计规范
- **统一结构**：色板 + 排版规则 + 组件样式 + 布局原则
- **预览 HTML**：每个设计系统配有视觉参考
- **分类组织**：AI 平台、开发工具、金融科技、零售、汽车等

## 使用方式

将对应品牌的 DESIGN.md 放入项目根目录，指示 AI 编码 Agent（Claude Code / Cursor / Codex）参考该文件生成匹配风格的 UI 组件。

## 为什么值得关注

- 61.3K Stars 反映"用 Markdown 描述设计系统"已成为社区共识
- 与 ui-design-brain、impeccable 等 Skill 互补，提供具体品牌参考
- 对独立开发者和小团队极其友好，无需自建设计系统
