> ⭐ N/A | Markdown (Skill) | https://github.com/anthropics/skills/tree/main/skills/pptx

## 项目简介

Anthropic 官方 Claude Code Skill，通过自然语言描述即可生成专业幻灯片，支持图表、布局和主题定制。

## 核心功能

- **自然语言生成幻灯片**：描述内容需求即可自动生成包含标题、正文和图表的演示文稿
- **图表与布局支持**：内置多种幻灯片布局模板，支持插入柱状图、饼图等数据图表
- **主题与样式定制**：可指定配色方案、字体和母版样式，保持企业品牌一致性

## 技术特点

- 基于 python-pptx 操作 OOXML 格式，生成原生 .pptx 文件
- 支持图片插入和占位符自动填充
- 与 XLSX Skill 联动，可直接从表格数据生成图表幻灯片

## 使用场景

- 快速将项目周报、数据分析结果转为演示文稿
- 批量生成培训课件或产品介绍 PPT
- 将 Markdown 大纲一键转换为带图表的幻灯片
