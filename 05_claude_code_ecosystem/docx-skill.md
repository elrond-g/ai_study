## 项目简介

Anthropic 官方 Claude Code Skill，用于创建和编辑 Word 文档，支持修订标记、批注及复杂格式排版。

## 核心功能

- **文档创建与编辑**：通过自然语言指令生成或修改 .docx 文件，保留样式和格式
- **修订标记与批注**：支持 Track Changes 模式，自动添加批注和修改建议
- **格式排版控制**：处理标题层级、表格、列表、页眉页脚等 Word 原生格式

## 技术特点

- 基于 python-docx 等库实现底层 OOXML 操作
- Skill 架构与 Claude Code 深度集成，支持对话式编辑
- 输出文件兼容 Microsoft Word、Google Docs 等主流办公软件

## 使用场景

- 根据模板批量生成合同、报告等标准化文档
- 对已有文档进行审阅并自动添加修订意见
- 将 Markdown 或纯文本内容转换为排版精良的 Word 文档
