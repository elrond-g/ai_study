## 项目简介

Anthropic 官方的画布设计 Skill，输入文字描述即可生成社交媒体配图、海报等视觉内容。将 Claude 变成一个文字驱动的平面设计工具。

## 核心功能

- **社交媒体图生成**：一键生成适配各平台尺寸的社交媒体配图
- **海报与宣传物料**：通过自然语言描述快速生成海报、Banner 等营销素材
- **排版与构图**：自动处理文字排版、元素布局和视觉层次

## 技术特点

- 使用 HTML Canvas / SVG 技术渲染输出
- 内置多种预设模板和布局方案
- 支持自定义配色、字体和品牌元素

## 使用场景

- 快速制作社交媒体推文配图
- 生成活动海报和宣传横幅
- 为博客文章和演示文稿创建封面图
