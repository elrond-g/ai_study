# agent-skills (Addy Osmani)

> ⭐ 6,500+ | Shell | https://github.com/addyosmani/agent-skills

## 项目简介

agent-skills 是 Google Chrome 团队 Addy Osmani 维护的面向 AI 编程 Agent 的生产级工程技能集合，适配 Claude Code / Cursor / Antigravity 等多个 AI 编程平台。

## 核心功能

- **生产级技能集**：覆盖开发工作流的多个关键环节
- **多平台适配**：支持 Claude Code、Cursor、Antigravity 等
- **工程化导向**：聚焦实际生产环境的开发技能

## 技术特点

- Shell 脚本为主
- 遵循主流 Agent Skill 规范
- 由 Google 工程师维护，质量有保障

## 使用场景

- AI Agent 编码能力增强
- 跨平台 Agent 技能复用
- 生产级开发流程自动化
