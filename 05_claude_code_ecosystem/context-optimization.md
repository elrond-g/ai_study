# Context Optimization

> ⭐ 1.39 万 | Markdown | https://github.com/muratcankoylan/agent-skills-for-context-engineering

## 项目简介

社区热门的上下文工程 Skill 集合，通过精细化 Token 管理和提示压缩技术显著降低 Claude Code 使用成本。

## 核心功能

- **Token 消耗优化**：自动压缩冗余上下文，减少每次对话的 Token 用量
- **上下文分层管理**：将系统提示、用户指令和代码上下文分层缓存，按需加载
- **成本监控与分析**：提供 Token 消耗统计和优化建议，帮助量化节省效果

## 技术特点

- 基于 Context Engineering 理念设计，专注上下文质量而非数量
- 兼容 Claude Code 原生 Skill 架构，开箱即用
- GitHub 1.39 万 stars，社区活跃度高，持续迭代更新

## 使用场景

- 大型代码仓库中降低 Claude Code 的 API 调用成本
- 长对话场景下保持上下文相关性同时减少 Token 消耗
- 团队级部署时优化整体 AI 辅助开发预算
