## 项目简介

社区维护的 Claude Skills 精选列表，分类齐全、持续更新。是发现和选择 Claude Skills 的一站式导航资源。

## 核心功能

- **分类索引**：按设计、开发、营销、知识管理等类别整理所有优质 Skill
- **持续更新**：社区驱动，定期收录新发布的高质量 Skill
- **质量筛选**：每个收录项目都经过社区评估，确保实用性和可靠性

## 技术特点

- 采用 Awesome List 标准格式，结构清晰易于浏览
- 每个条目包含简介、链接和标签，便于快速筛选
- 开放社区贡献，通过 PR 机制保证收录质量

## 使用场景

- 查找特定领域的 Claude Skill 推荐
- 了解 Claude Skills 生态的整体概况和发展趋势
- 作为开发新 Skill 时的灵感来源和参考
