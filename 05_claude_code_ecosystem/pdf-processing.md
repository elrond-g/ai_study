# PDF Processing

> ⭐ N/A | Markdown (Skill) | https://github.com/anthropics/skills/tree/main/skills/pdf

## 项目简介

Anthropic 官方 Claude Code Skill，专注于 PDF 文档处理，支持读取、解析、表格提取、表单填写以及合并拆分等常见操作。

## 核心功能

- **PDF 读取与解析**：自动识别文档结构，提取文本、图片和元数据
- **表格提取**：精准定位 PDF 中的表格内容并转换为结构化数据
- **表单填写与合并拆分**：支持程序化填写 PDF 表单字段，以及多文档合并和按页拆分

## 技术特点

- 基于 Claude Code Skill 架构，通过 SKILL.md 声明式配置
- 利用 Claude 多模态能力处理扫描件和复杂排版
- 与其他 Office 类 Skill（DOCX、XLSX）可组合使用

## 使用场景

- 批量提取合同、发票中的关键字段和表格数据
- 自动化填写政府或企业标准 PDF 表单
- 将多份报告合并为单一 PDF 或按章节拆分归档
