# Marketing Skills

> ⭐ N/A | JavaScript | https://github.com/coreyhaines31/marketingskills

## 项目简介

全方位营销 Skill 套件，内含 CRO（转化率优化）、文案撰写、SEO、邮件营销、增长策略等 20+ 子技能。为营销人员提供一站式 AI 辅助工作流。

## 核心功能

- **SEO 优化**：关键词研究、内容优化、元数据生成等搜索引擎优化能力
- **文案创作**：广告文案、落地页文案、社交媒体内容等多场景文案生成
- **增长策略**：转化率分析、A/B 测试方案、用户增长漏斗优化建议

## 技术特点

- 模块化设计，20+ 子技能可独立或组合使用
- 覆盖营销全链路：获客、转化、留存、传播
- 内置行业最佳实践和营销方法论模板

## 使用场景

- 创业团队快速搭建营销内容体系
- 优化网站转化率和搜索引擎排名
- 批量生成多渠道营销物料和邮件序列
