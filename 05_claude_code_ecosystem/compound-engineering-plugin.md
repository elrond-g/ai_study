# Compound Engineering Plugin

> ⭐ 13,400+ | TypeScript | https://github.com/EveryInc/compound-engineering-plugin

## 项目简介

Compound Engineering 官方插件，推行 80% 规划 + 20% 执行的复利工程方法论。提供 brainstorm/plan/work/review/compound 全流程 Agent 技能，支持 Claude Code / Cursor / OpenCode 等 10+ AI 编程平台。

## 核心功能

- **复利工程方法论**：80% 规划 + 20% 执行的结构化工作流
- **全流程技能**：brainstorm → plan → work → review → compound
- **多平台支持**：Claude Code、Cursor、OpenCode 等 10+ 平台
- **知识复利**：每次开发积累可复用的工程知识

## 技术特点

- TypeScript 实现
- 遵循 Agent Skill 标准规范
- 支持跨平台技能迁移

## 使用场景

- 结构化的 AI 辅助开发流程
- 工程知识的持续积累与复用
- 多平台 Agent 技能标准化
