## 项目简介

Anthropic 官方的主题工厂 Skill，一个 prompt 即可批量生成完整的配色方案。帮助开发者和设计师快速产出一致性高的主题系统。

## 核心功能

- **批量配色生成**：一次生成亮色、暗色等多套配色方案
- **色彩系统构建**：自动生成主色、辅色、中性色等完整色阶
- **CSS 变量输出**：直接输出可用的 CSS 自定义属性和设计 Token

## 技术特点

- 基于色彩理论自动计算对比度，确保无障碍可读性
- 支持 HSL、RGB、Hex 等多种色彩格式输出
- 生成的主题可直接集成到 Tailwind CSS 或 CSS-in-JS 方案

## 使用场景

- 为新项目快速确定品牌配色体系
- 批量生成产品的亮色/暗色主题切换方案
- 设计系统中的色彩 Token 管理和文档生成
