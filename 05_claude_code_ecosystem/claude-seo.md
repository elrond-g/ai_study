## 项目简介

专注于 SEO 审计和优化的 Claude Skill，支持全站 SEO 诊断和 Schema 结构化数据验证。帮助网站快速发现并修复搜索引擎优化问题。

## 核心功能

- **全站 SEO 审计**：自动扫描网站页面，检测标题、描述、链接等 SEO 要素
- **Schema 验证**：验证 JSON-LD 结构化数据的正确性和完整性
- **优化建议生成**：针对发现的问题提供具体的修复方案和优先级排序

## 技术特点

- 覆盖技术 SEO、内容 SEO、站内链接等多维度检查
- 支持 Schema.org 标准的结构化数据校验
- 输出结构化审计报告，便于团队协作和追踪

## 使用场景

- 网站上线前的 SEO 健康检查
- 定期执行全站 SEO 审计，持续优化搜索排名
- 验证和生成符合 Google 规范的结构化数据标记
