# Tianji（天机）

> ⭐ 1,723 | Python | https://github.com/SocialAI-tianji/Tianji | https://socialai-tianji.github.io/socialai-web/

## 项目简介

天机（Tianji）是一个让大语言模型"更懂人情世故"的项目，涵盖提示词工程、RAG、Agent 和 LLM 微调的完整教程与实践。项目目标是训练出能够理解中国社交语境、处理人际关系场景的 AI 助手。

## 核心功能

- **人情世故场景库**：收录大量中国社交场景的对话数据（送礼、职场沟通、家庭矛盾等）
- **Prompt 工程教程**：从基础到高级的提示词设计指南
- **RAG 实践**：检索增强生成的完整实现示例
- **Agent 开发**：基于 LangChain 的多工具 Agent 构建
- **LLM 微调**：Qwen 等中文模型的微调教程（含数据准备）

## 技术特点

- Python + LangChain 技术栈
- 支持 Qwen 等中文开源模型
- 提供完整的数据集和微调脚本

## 使用场景

- 开发针对中国用户的 AI 助手，需要理解本地社交语境
- 学习从零到一的 LLM 应用开发全链路
- 研究如何让模型输出更具人情味和社交智慧
