# MemOS

> https://github.com/MemTensor/MemOS

## 项目简介

MemOS 是一个上下文管理操作系统（Memory OS），将操作系统的内存管理理念引入 AI Agent 的上下文管理。像 OS 统一管理物理内存一样，MemOS 统一管理 AI Agent 的各类"记忆"——工作记忆、长期记忆、外部知识——并提供标准化的访问接口。

## 核心功能

- **统一记忆抽象**：工作记忆（上下文窗口）、短期记忆（会话历史）、长期记忆（持久存储）统一管理
- **动态上下文分配**：根据任务需求动态调整上下文内容，类似 OS 的虚拟内存机制
- **记忆检索**：基于语义相关性从长期存储中召回相关记忆
- **插件架构**：可插拔的存储后端，支持向量数据库、图数据库等

## 技术特点

- 操作系统隐喻简化了复杂记忆管理的理解
- 与主流向量数据库（Chroma、Weaviate 等）集成
- 提供 Python SDK，对接现有 AI Agent 框架

## 使用场景

- 构建需要长期记忆的 AI 助手（客服机器人、个人助理等）
- 管理超长任务中 Agent 的上下文使用效率
- 研究 AI 记忆管理的系统级架构
