# vobase

> https://github.com/vobase/vobase

## 项目简介

vobase 是一个向量存储基础服务，为 AI 应用提供高效的向量数据管理能力。专注于向量的存储、索引和检索，作为 RAG 系统和 AI 记忆管理的底层存储层。

## 核心功能

- **向量存储**：高效存储高维向量数据（文本/图像嵌入等）
- **相似度检索**：余弦相似度、欧式距离等多种距离度量
- **批量操作**：支持大规模向量的批量插入和检索
- **元数据过滤**：结合向量相似度和结构化元数据的混合查询

## 技术特点

- 轻量化设计，部署简单
- 提供 REST API，语言无关
- 支持持久化存储和内存模式

## 在 AI 记忆中的角色

vobase 在 AI 应用中通常作为底层存储层：
```
用户输入 → 嵌入模型 → 向量 → vobase（存储/检索）→ 注入 LLM 上下文
```

## 使用场景

- RAG 系统的向量存储后端
- AI 记忆系统的持久化层
- 文档语义搜索服务
