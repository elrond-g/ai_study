# context-hub

> https://github.com/andrewyng/context-hub

## 项目简介

context-hub 是 AI 教育领域知名人物 Andrew Ng 的上下文管理项目，提供一个标准化的"上下文中心"，集中管理 AI 应用开发中各类可复用的上下文组件——系统提示词、工具描述、知识片段、示例对话等。

## 核心概念

上下文管理的核心挑战：
- 好的 Prompt 和上下文难以复用
- 不同 AI 工具使用不同格式，迁移成本高
- 团队协作时上下文难以版本控制和共享

context-hub 的解决思路：
- 将上下文组件作为独立资产管理
- 标准化格式，跨工具复用
- 支持版本控制和协作分享

## 核心功能

- **上下文组件库**：系统提示词、Few-shot 示例、工具描述的结构化存储
- **版本管理**：追踪上下文的迭代历史
- **跨工具兼容**：转换为不同 AI 工具的格式
- **协作共享**：团队内部共享和复用高质量上下文

## 使用场景

- 企业建立组织级别的 AI 上下文知识库
- 开发者系统化管理和复用 Prompt 资产
- 研究 Andrew Ng 对 AI 工程实践的理解
