# oh-my-codex (OMX)

> ⭐ 17,600+ | TypeScript | https://github.com/Yeachan-Heo/oh-my-codex

## 项目简介

oh-my-codex（OMX）是 OpenAI Codex CLI 的工作流增强层，提供 deep-interview/ralplan/team/ralph 等标准化工作流，支持多 Agent 并行执行与持久化状态管理。

## 核心功能

- **标准化工作流**：deep-interview、ralplan、team、ralph 等预设流程
- **多 Agent 并行**：支持同时运行多个 Agent 实例
- **持久化状态**：跨会话状态管理和恢复
- **Codex CLI 增强**：在原有能力上叠加工作流层

## 技术特点

- TypeScript 实现
- 构建于 OpenAI Codex CLI 之上
- 标准化工作流模板

## 使用场景

- OpenAI Codex CLI 能力增强
- 结构化的 AI 辅助开发流程
- 多 Agent 协作开发
