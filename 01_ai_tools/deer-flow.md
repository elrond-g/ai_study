# deer-flow

> ⭐ 41,211 | Python | https://github.com/bytedance/deer-flow

## 项目简介

字节跳动开源的 SuperAgent 框架，将研究、编码、创作能力集于一体。借助沙盒、记忆、工具、Skills、子 Agent 和消息网关，可处理从几分钟到数小时不等的复杂任务。

## 核心功能

- **研究能力**：自主搜索、阅读、归纳信息，生成结构化报告
- **编码能力**：在沙盒中安全执行代码，调试和迭代
- **子 Agent 编排**：将大任务拆分给专用子 Agent 并行处理
- **记忆系统**：跨会话保留上下文和中间结果
- **消息网关**：对接外部通信渠道（微信、飞书等）
- **Skills 支持**：通过 Skill 文件扩展 Agent 专业能力

## 技术特点

- Python 实现，模块化架构，各能力层解耦
- 沙盒隔离执行环境，代码运行安全可控
- 支持多种工具接入（浏览器、文件系统、外部 API）
- 与 Claude、GPT 等主流模型兼容

## 使用场景

- 长周期自动化研究任务（竞品分析、技术调研）
- 需要编码+验证+报告一体化的复杂工作流
- 企业内部 AI 助理，通过消息网关接入现有 IM 工具
