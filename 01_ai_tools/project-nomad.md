# Project N.O.M.A.D

> ⭐ 24.6k | Apache 2.0 | https://github.com/Crosstalk-Solutions/project-nomad

## 项目简介

离线优先的知识与教育服务器，安装完成后无需互联网即可完整运行。将 AI 聊天、离线维基、医学参考、教育课程、地图、加解密工具等整合在 Docker 容器中，一次部署即可在断网环境下提供完整知识基础设施。

## 集成服务

- **AI Chat**：基于 Ollama 的本地对话 + Qdrant 向量库支持文档上传和语义搜索
- **离线维基**：通过 Kiwix 提供离线 Wikipedia、医学参考、电子书
- **教育课程**：Khan Academy 课程（通过 Kolibri 平台）
- **地图**：ProtoMaps 可下载离线地图
- **安全工具**：CyberChef 数据加密与分析
- **笔记应用**：本地笔记系统
- **硬件基准**：社区排行榜分享部署性能

## 技术栈

- Docker 容器化架构
- TypeScript 92.3% + Shell 脚本
- 基于 Debian 系统部署

## 特色

- **零 Telemetry**：彻底隐私保护
- **零认证**：设计上假设本地可信环境
- **离线优先**：所有核心功能断网可用

## 使用场景

- 偏远地区教育资源部署
- 应急/灾难响应场景
- 完全脱网的个人知识中心
- 数字主权实验
