# DeepTutor

> ⭐ 11,500+ | Python | https://github.com/HKUDS/DeepTutor

## 项目简介

DeepTutor 是港大 HKUDS 团队出品的 Agent 原生个性化 AI 辅导系统，集成 Chat/深度求解/测验生成/深度研究/数学动画五种模式，支持自主进化 TutorBot 和持久记忆。

## 核心功能

- **五模式教学**：Chat、深度求解、测验生成、深度研究、数学动画
- **个性化辅导**：根据学生水平和偏好自适应调整
- **自主进化 TutorBot**：Agent 可自我改进教学策略
- **持久记忆**：跨会话记录学习进度和知识掌握情况

## 技术特点

- Python 实现
- Agent 原生架构
- 支持多种学科领域

## 使用场景

- AI 个性化教育辅导
- 自适应学习系统
- 学术研究和自学辅助
