# Firecrawl

> ⭐ 25k+ | TypeScript / Rust | https://github.com/mendableai/firecrawl

## 项目简介

Firecrawl 是一个专为 LLM 设计的网页爬取引擎，能够将任意网页内容转换为干净的 Markdown 或结构化数据，为 AI 应用提供高质量的网页数据输入。

## 核心功能

- **智能内容提取**：自动去除广告、导航等噪音，提取网页核心内容
- **多格式输出**：支持输出为 Markdown、HTML、纯文本和结构化 JSON
- **批量爬取与站点地图**：支持整站爬取，自动发现子页面并批量处理

## 技术特点

- 内置无头浏览器渲染，能处理 JavaScript 动态加载页面
- 提供 REST API 和多语言 SDK（Python、Node.js、Go、Rust）
- 支持自托管和云服务两种部署方式

## 使用场景

- 为 RAG 系统提供高质量网页数据源
- LLM 应用中的实时网页信息获取
- 构建知识库时的大规模网页内容采集
