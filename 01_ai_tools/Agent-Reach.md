# Agent-Reach

> Panniantong | Python CLI | https://github.com/Panniantong/Agent-Reach

## 项目简介

Agent-Reach 为 AI Agent 赋予"网络视觉"，通过统一 CLI 工具零 API 费用访问 16 个社交平台和网站。使用 Cookie 认证而非官方 API，覆盖国际和中文互联网平台。

## 支持的 16 个平台

**国际**：Twitter/X、Reddit、YouTube、GitHub、LinkedIn、Instagram
**中文**：小红书、抖音、微博、微信公众号、小宇宙播客
**其他**：V2EX、RSS 源、Exa 网络搜索、任意 URL

## 核心功能

- **搜索**：平台内容搜索
- **阅读**：原文内容获取
- **下载**：元数据和字幕
- **转录**：视频/音频转录文本
- **时间线**：帖子线程和用户时间线
- **爬取**：网页爬取和解析

## 技术特点

- **零 API 费用**：Cookie 认证方案，Twitter 访问用 xreach CLI
- **环境自适应**：Agent 自动安装、检测环境并报告可用功能
- **Claude Code 集成**：可作为 Skill 安装

## 安装

```bash
pip install https://github.com/Panniantong/agent-reach/archive/main.zip
```

## 为什么值得关注

1. 零 API 费用的多平台统一访问对大规模 Agent 部署有重要价值
2. 特别覆盖中文互联网平台（小红书、微博、抖音等），填补英文工具空白
3. 统一接口设计支持新平台扩展
