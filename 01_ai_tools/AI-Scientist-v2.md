# AI-Scientist-v2

> ⭐ 5,100+ | Python | https://github.com/SakanaAI/AI-Scientist-v2

## 项目简介

AI-Scientist-v2 是 Sakana AI 推出的全自动科研系统第二版，通过 Agent 树搜索自主完成假设生成、实验执行、数据分析和论文撰写全流程。已产出首篇被同行评审接收的 AI 独立撰写 Workshop 论文。

## 核心功能

- **全自动科研流程**：从假设提出到论文撰写的端到端自动化
- **Agent 树搜索**：渐进式搜索策略替代人工模板
- **实验执行**：自主设计并执行计算实验
- **数据分析**：自动分析实验结果并生成可视化
- **论文生成**：输出符合学术规范的完整论文

## 技术特点

- v1 的进化版，移除了人工模板依赖
- 采用渐进式 Agent 树搜索算法
- Python 实现，集成多种科研工具链

## 使用场景

- AI for Science 自动化研究
- 科研论文自动撰写
- 实验设计与执行的自动化探索
