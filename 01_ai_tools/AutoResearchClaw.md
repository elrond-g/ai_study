# AutoResearchClaw

> ⭐ 7,800+ | Python | https://github.com/aiming-lab/AutoResearchClaw

## 项目简介

AutoResearchClaw 是一个完全自主、自进化的 AI 科研 Agent，能够从一个研究想法出发，自动完成文献调研、实验设计、代码实现、结果分析直到最终生成论文的全流程。口号是"Chat an Idea. Get a Paper."

## 核心功能

- **端到端科研自动化**：从 Idea 到论文，全程 AI 驱动
- **自进化能力**：Agent 能从历次研究中学习，优化下一次研究策略
- **多 Agent 辩论**：内置多智能体辩论机制，提升研究结论的严谨性
- **引文验证**：自动验证引用文献的真实性，防止幻觉引用
- **自定义研究方向**：支持设定研究领域约束和目标

## 技术特点

- 基于 OpenClaw 生态构建，与 Claude Code 深度集成
- 支持 MetaClaw 框架进行 Agent 级别的元学习
- 论文生成遵循学术规范（摘要、方法、实验、结论）

## 使用场景

- 研究人员加速论文产出，聚焦核心思路而非执行细节
- 探索 AI 驱动的科学发现工作流
- 学习多 Agent 协作在专业领域的落地方案
