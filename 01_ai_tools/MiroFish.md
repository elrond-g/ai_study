# MiroFish

> ⭐ 40,500+ | Python | https://github.com/666ghj/MiroFish | https://mirofish.ai

## 项目简介

MiroFish 是一个简洁通用的群体智能引擎，用于预测金融走势、社会舆情和世界事件。基于多 Agent 模拟和知识图谱，它让多个 AI "鱼群"从不同视角分析同一问题，汇聚成集体判断。

## 核心功能

- **群体智能预测**：多 Agent 协作模拟，输出比单一模型更稳健的预测
- **金融市场分析**：股票、加密货币等金融资产走势预测
- **舆情趋势判断**：公众情绪分析与事件走向预测
- **知识图谱集成**：构建和利用领域知识图谱增强推理能力
- **长期记忆**：Agent 在预测过程中积累的知识可跨轮次复用

## 技术特点

- Python 实现，模块化架构
- 多 Agent 仿真框架，支持自定义 Agent 角色和交互规则
- 知识图谱 + LLM 双引擎驱动

## 使用场景

- 量化研究人员构建 AI 辅助的市场预测系统
- 社会科学研究者模拟群体行为
- 探索集体智慧在 AI 预测中的应用
