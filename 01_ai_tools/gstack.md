# gstack

> ⭐ 40,000+ | TypeScript | https://github.com/garrytan/gstack

## 项目简介

gstack 是 Y Combinator CEO（总裁）Garry Tan 的 Claude Code 工作流开源版本。它提供 15 个高度定制化的 AI 工具，模拟真实创业公司的核心职能——CEO 决策、设计师审美、工程经理调度、发布管理、文档工程和 QA 测试——用于加速个人或小团队的软件开发流程。

## 核心功能

- **15 个专业 Agent 角色**，覆盖完整的软件交付链路
- **CEO Agent**：从商业视角审视技术决策
- **Designer Agent**：执行 UI/UX 审查，确保视觉一致性
- **Engineering Manager Agent**：拆解任务、评估优先级
- **Release Manager Agent**：管理版本发布流程
- **Doc Engineer Agent**：自动生成和维护技术文档
- **QA Agent**：编写和执行测试用例

## 技术特点

- 基于 TypeScript，与 Claude Code 生态无缝集成
- 每个角色都有精心打磨的系统提示词，体现创业公司实战经验
- 从不同角度对同一问题发起多轮审查，降低单点失误

## 使用场景

- 独立开发者或小团队希望引入"虚拟 CTO/CPO"辅助决策
- 在 Claude Code 中快速切换不同职能视角来 review 代码
- 学习如何设计高质量的专业角色 prompt
