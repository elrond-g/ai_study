# GitNexus

> ⭐ 23,600+ | TypeScript | https://github.com/abhigyanpatwari/GitNexus

## 项目简介

GitNexus 是一个零服务端的代码知识图谱引擎，将代码库索引为交互式知识图谱（依赖关系/调用链/执行流），内置 Graph RAG Agent，支持 CLI+MCP 与浏览器 Web UI 双模式。

## 核心功能

- **代码知识图谱**：自动构建依赖关系、调用链、执行流图谱
- **Graph RAG Agent**：基于知识图谱的智能代码问答
- **零服务端**：本地运行，无需后端服务
- **双模式交互**：CLI+MCP 和浏览器 Web UI

## 技术特点

- TypeScript 实现
- 知识图谱驱动的代码理解
- MCP 协议原生支持

## 使用场景

- 大型代码库的结构可视化和理解
- AI Agent 的代码导航和问答
- 代码审查和架构分析
