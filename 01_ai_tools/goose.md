# Goose

> ⭐ 38,100+ | Rust | https://github.com/aaif-goose/goose

## 项目简介

Goose 是一个开源可扩展 AI Agent，超越代码建议——能自主安装依赖、执行代码、编辑文件和运行测试。支持任意 LLM 后端和 MCP 集成，提供桌面端和 CLI 两种使用方式。原由 Block (Square) 出品，现迁移至 aaif-goose 组织。

## 核心功能

- **自主执行**：不只是建议代码，能自主完成完整开发任务
- **多模型支持**：支持配置任意 LLM（Claude/GPT/Gemini/本地模型等）
- **MCP 集成**：支持 MCP 服务器扩展工具能力
- **桌面端 + CLI**：提供图形界面和命令行两种交互方式
- **工作流编排**：自动构建完整项目、调试和工作流管理

## 技术特点

- Rust 核心实现，高性能
- 可扩展的插件/工具系统
- 支持沙箱和安全执行环境

## 使用场景

- AI 辅助编码与自主开发
- 项目从零搭建和调试
- 替代或补充 Claude Code 等工具的编码 Agent
