# OpenClaw

> ⭐ 210k+ | Python | https://github.com/openclaw/openclaw

## 项目简介

OpenClaw 是一个社区驱动的开源通用 AI Agent 基础设施框架，拥有超过 21 万 stars，是目前最受关注的 Agent 开源项目之一。它提供了构建、部署和管理自主 Agent 的完整工具链。

## 核心功能

- **通用 Agent 构建**：提供灵活的 Agent 抽象层，支持快速搭建各类自主智能体
- **社区插件生态**：丰富的社区贡献插件和工具集成，覆盖搜索、代码、数据等场景
- **多模型支持**：兼容主流 LLM 提供商，支持模型热切换和负载均衡

## 技术特点

- 模块化架构设计，核心组件可独立替换和扩展
- 内置任务编排引擎，支持复杂多步骤工作流
- 提供标准化的 Agent 通信协议，便于多 Agent 协作

## 使用场景

- 构建企业级自主 Agent 应用
- 快速原型验证 Agent 方案
- 作为 AI Agent 研究和教学的基础平台
