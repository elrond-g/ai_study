# AutoGPT

> ⭐ 170k+ | Python | https://github.com/Significant-Gravitas/AutoGPT

## 项目简介

AutoGPT 是自主 AI Agent 领域的先驱项目，能够根据用户设定的高层目标自动进行任务分解，并通过多步骤迭代执行来完成复杂任务，无需人工干预每一步操作。

## 核心功能

- **自主目标分解**：将高层目标自动拆解为可执行的子任务链
- **工具调用与执行**：内置网页浏览、文件操作、代码执行等多种工具能力
- **记忆与上下文管理**：支持短期和长期记忆，确保多步骤任务的上下文连贯性

## 技术特点

- 基于 LLM 的自主推理循环（思考-计划-执行-反馈）
- 插件化架构，支持自定义工具和第三方服务集成
- 提供 Forge 框架用于快速构建自定义 Agent

## 使用场景

- 自动化市场调研和信息收集
- 代码生成与项目脚手架搭建
- 多步骤业务流程的自主执行
