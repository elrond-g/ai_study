# knowledge-work-plugins

> ⭐ 10,000+ | Python | https://github.com/anthropics/knowledge-work-plugins

## 项目简介

knowledge-work-plugins 是 Anthropic 官方为知识工作者发布的开源插件集合，主要面向 Claude Cowork 平台使用。每个插件对应一类专业知识工作者的具体工作场景，为 AI 提供精确的角色定义、工作流程和输出规范。

## 核心功能

- **官方认证的专业角色**：Anthropic 直接维护，质量有保障
- **覆盖多个知识工作领域**：法律、医疗、金融、教育、工程等
- **标准化插件接口**：统一的格式，方便开发者按需加载
- **可组合使用**：多个插件可以组合，模拟跨职能团队协作

## 技术特点

- Python 实现，符合 Anthropic 插件规范
- 插件描述结构化，包含角色定位、能力范围和边界约束
- 与 Claude API 深度集成，可通过系统提示词直接激活

## 使用场景

- 企业为不同部门员工定制专业 AI 助手
- 开发者基于官方模板快速构建垂直领域 AI 工具
- 研究如何为专业角色设计高质量 AI 插件
