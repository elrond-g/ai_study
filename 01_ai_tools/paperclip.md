# paperclip

> ⭐ 31,000+ | TypeScript | https://github.com/paperclipai/paperclip | https://paperclip.ing

## 项目简介

paperclip 是"0 人公司"（zero-human company）概念的开源实现，提供完整的 AI 代理编排框架。项目的目标是让 AI Agent 自主运营一家公司：接受任务、拆解需求、分配给子 Agent 执行、汇总结果，全程无需人工干预。

## 核心功能

- **自主任务编排**：接收高层目标，自动分解为可执行的子任务序列
- **多 Agent 协同**：内置任务分发机制，多个专项 Agent 并行处理
- **状态跟踪**：实时监控每个子任务的执行状态和结果
- **可配置角色**：支持自定义 Agent 角色及其职责边界

## 技术特点

- TypeScript 实现，具备类型安全和良好的 IDE 支持
- 模块化架构，可按需扩展新的 Agent 类型
- 内置失败重试和错误处理机制

## 使用场景

- 探索和验证全自动化软件交付可行性
- 构建自定义的多 Agent 业务流程自动化
- 研究 AI 驱动的公司运营模型
