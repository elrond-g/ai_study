# AstrBot

> ⭐ 26,679 | Python | https://github.com/AstrBotDevs/AstrBot | https://astrbot.app

## 项目简介

AstrBot 是一个功能完整的 Agentic IM 聊天机器人基础设施，支持接入微信、QQ、Telegram、Discord 等主流即时通讯平台，并集成 LLM、插件系统和 AI 功能。可作为 OpenClaw（Claude）的替代方案，在 IM 环境中运行 AI Agent。

## 核心功能

- **多平台接入**：QQ、微信、Telegram、Discord 等平台统一管理
- **LLM 集成**：支持 ChatGPT、Claude、Gemini、Llama 等多个模型
- **MCP 支持**：通过 MCP 协议扩展工具调用能力
- **插件系统**：丰富的社区插件，可自由扩展功能
- **Docker 部署**：容器化部署，5 分钟快速上线

## 技术特点

- Python 异步架构，高并发处理消息
- 统一的消息抽象层，屏蔽各平台 API 差异
- 完整的管理后台，可视化配置和监控
