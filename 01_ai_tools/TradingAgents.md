# TradingAgents

> ⭐ 47,900+ | Python | https://github.com/TauricResearch/TradingAgents

## 项目简介

TradingAgents 是一个多 Agent LLM 金融交易框架，模拟真实交易公司的多角色协作机制（分析师、交易员、风控经理等），利用大语言模型进行智能投资决策。配套学术论文 arXiv:2412.20138。

## 核心功能

- **多角色 Agent 协作**：分析师、交易员、风控经理等角色分工明确
- **市场分析**：自动分析市场数据和新闻信息
- **交易决策**：基于多 Agent 讨论生成交易建议
- **风险控制**：内置风控 Agent 审查交易决策
- **回测支持**：支持历史数据回测验证策略

## 技术特点

- Python 实现，基于 LangGraph 多 Agent 编排
- 支持多种 LLM 后端
- 模拟交易公司的层级决策流程

## 使用场景

- LLM 驱动的量化交易研究
- 金融市场分析与策略生成
- 多 Agent 协作在金融领域的应用探索
