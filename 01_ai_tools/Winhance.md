# Winhance

> ⭐ 8,100+ | C# | https://github.com/memstechtips/Winhance

## 项目简介

Winhance 是一个 Windows 10/11 系统优化与精简工具，提供自定义、去臃肿和性能增强功能，帮助用户清理系统预装应用并优化系统设置。

## 核心功能

- **系统精简**：移除 Windows 预装臃肿软件
- **性能增强**：优化系统设置提升运行速度
- **自定义配置**：灵活调整 Windows 系统行为
- **安全操作**：可逆操作，支持还原

## 技术特点

- C# 实现
- 支持 Windows 10 和 Windows 11
- 图形界面操作

## 使用场景

- Windows 系统初始化清理
- 老旧硬件性能优化
- 开发环境系统精简
