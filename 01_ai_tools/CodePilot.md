# CodePilot

> ⭐ 4,447 | TypeScript | https://github.com/op7418/CodePilot | https://www.codepilot.sh/

## 项目简介

CodePilot 是 Claude Code 的桌面 GUI 封装，基于 Electron + Next.js 构建。它将 Claude Code 强大的 AI 编程能力包装进可视化界面，让不习惯纯命令行的开发者也能高效使用 Claude Code 的全部功能。

## 核心功能

- **可视化对话界面**：以聊天形式与 Claude Code 交互，支持富文本显示
- **项目管理面板**：图形化管理多个项目，快速切换工作目录
- **文件树集成**：实时显示项目文件结构，支持可视化浏览
- **会话历史**：保存和回顾历史对话记录

## 技术特点

- Electron + Next.js 技术栈，跨平台桌面应用
- 底层调用 Claude Code CLI，保留全部 AI 能力
- 桌面原生体验，支持快捷键和系统托盘

## 使用场景

- 不熟悉命令行但想使用 Claude Code 的开发者
- 需要同时管理多个 AI 编程项目的团队
- 希望将 Claude Code 集成进桌面工作流的用户
