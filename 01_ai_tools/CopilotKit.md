# CopilotKit

> ⭐ 15k+ | TypeScript | https://github.com/CopilotKit/CopilotKit

## 项目简介

CopilotKit 是一个专为 React 应用设计的开源 AI Copilot 集成框架，让开发者能够快速为现有应用添加智能助手能力，支持自定义 Agent、上下文感知和用户交互动作。

## 核心功能

- **React 组件库**：提供即用的 Copilot 侧边栏、弹窗、文本补全等 UI 组件
- **自定义动作（Actions）**：允许 Copilot 调用开发者定义的前端/后端函数
- **Agent 集成**：支持接入 LangGraph 等框架的 Agent，实现复杂交互流程

## 技术特点

- 通过 React Hooks 和 Context 实现应用状态与 AI 的深度绑定
- 支持流式输出和实时协作编辑
- 前后端 SDK 分离，后端支持 Node.js 和 Python

## 使用场景

- 为 SaaS 产品添加 AI 辅助功能
- 构建具有自然语言操控能力的管理后台
- 在编辑器类应用中集成智能写作助手
