# Microsoft Agent Framework

> ⭐ 9,000+ | Python / .NET | https://github.com/microsoft/agent-framework

## 项目简介

Microsoft Agent Framework 是微软官方开源的多语言 Agent 框架，支持构建、编排和部署单/多 Agent AI 工作流。提供图编排引擎，Python 和 .NET 双版本实现。

## 核心功能

- **多 Agent 编排**：基于图编排引擎的 Agent 工作流管理
- **双语言支持**：Python 和 .NET 两种语言 SDK
- **聊天 Agent**：开箱即用的对话式 Agent
- **工具集成**：灵活的工具和能力扩展接口
- **Agent 生命周期管理**：完整的创建、运行、监控、终止流程

## 技术特点

- 微软官方出品，企业级质量保证
- 替代 Semantic Kernel 中的 Agent 能力
- 图编排引擎支持复杂工作流拓扑

## 使用场景

- 企业级多 Agent AI 应用开发
- 复杂任务的 Agent 编排与协作
- .NET 生态的 AI Agent 集成
