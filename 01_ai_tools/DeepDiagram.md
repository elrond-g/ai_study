# DeepDiagram

> ⭐ 1,235 | TypeScript | https://github.com/twwch/DeepDiagram | http://deepd.cturing.cn

## 项目简介

DeepDiagram 是一个开源 AI 可视化工具，将自然语言描述实时转化为思维导图、Mermaid 流程图和 Echarts 图表。用户只需输入想法或需求，即可获得结构化的可视化表达。

## 核心功能

- **自然语言转图表**：支持思维导图、流程图、时序图、饼图等多种类型
- **Mermaid 支持**：生成标准 Mermaid 语法，可直接嵌入 Markdown 文档
- **Echarts 集成**：生成可交互的数据可视化图表
- **实时预览**：输入即时渲染，所见即所得

## 技术特点

- TypeScript + Next.js 全栈
- 接入主流 LLM API（支持 OpenAI、Claude 等）
- 支持一键导出为 PNG、SVG 或 Markdown

## 使用场景

- 快速将会议记录或脑暴内容转为思维导图
- 技术文档中自动生成架构图或流程图
- 用自然语言描述数据关系，生成可视化图表

## 快速开始

```bash
git clone https://github.com/twwch/DeepDiagram
pnpm install && pnpm dev
```
