# wechat-article-exporter

> ⭐ 8,030 | TypeScript | https://github.com/wechat-article/wechat-article-exporter

## 项目简介

微信公众号文章批量下载工具。无需搭建任何本地环境，可通过在线网站直接使用，也支持 Docker 私有化部署和 Cloudflare Pages 部署。

## 核心功能

- **批量下载**：一键抓取公众号全部历史文章
- **多格式导出**：HTML（100% 还原排版与样式）、Markdown、PDF 等
- **数据导出**：支持导出阅读量、点赞数、评论数等互动数据
- **私有化部署**：支持 Docker 自托管，数据不经第三方
- **Cloudflare 部署**：无服务器部署方案，零成本运行

## 技术特点

- TypeScript + 在线 SaaS 模式，零依赖直接使用
- HTML 导出保留原始 CSS，排版与微信端完全一致
- Docker 镜像开箱即用，私有化部署简单

## 使用场景

- 公众号内容备份与迁移
- 批量采集文章用于知识库建设或 RAG
- 导出互动数据进行内容分析
