# Magic Resume

> GitHub | https://github.com/JOYCEQL/magic-resume

## 项目简介

Magic Resume 是一个免费开源的 AI 在线简历编辑器，使用 TanStack Start 和 Framer Motion 构建，支持实时预览和自定义主题，创建专业简历简单高效。3,900+ Stars。

## 核心功能

- 🚀 基于 TanStack Start 构建
- 💫 流畅动画（Framer Motion）
- 🎨 自定义主题支持
- 📱 响应式设计
- 🌙 暗色模式
- 📤 导出 PDF
- 🔄 实时预览
- 💾 自动保存
- 🔒 本地存储

## 技术栈

- TanStack Start
- TypeScript
- Motion（Framer Motion）
- Tiptap（富文本编辑器）
- Tailwind CSS
- Zustand（状态管理）
- Shadcn/ui（组件库）
- Lucide Icons

## 快速开始

```bash
git clone git@github.com:JOYCEQL/magic-resume.git
cd magic-resume
pnpm install
pnpm dev
```

支持 Docker Compose 部署。
