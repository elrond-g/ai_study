# FileEngine

> ⭐ 39 | Go | https://github.com/huangzheng2016/FileEngine

## 项目简介

FileEngine 是一个 AI 驱动的文件整理引擎，通过 LLM Agent 扫描远程文件系统（Local/SFTP/FTP/SMB/NFS），自动标记分类并生成整理方案，支持多模型和并发 Agent。

## 核心功能

- **多文件系统支持**：Local/SFTP/FTP/SMB/NFS
- **AI 自动分类**：LLM 驱动的文件标记和分类
- **整理方案生成**：自动生成文件整理建议
- **并发 Agent**：支持多 Agent 并行扫描处理

## 技术特点

- Go 语言实现
- 支持多种远程文件系统协议
- 多 LLM 模型适配

## 使用场景

- 大量文件的自动分类整理
- NAS/服务器文件管理自动化
- 跨存储系统的统一文件治理
