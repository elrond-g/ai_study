# RuView

> ⭐ 44,000+ | Rust | https://github.com/ruvnet/RuView | MIT

## 项目简介

RuView 是一个 WiFi DensePose 系统，将普通消费级 WiFi 信号转化为实时人体姿态估计、生命体征监测和存在检测能力，完全无需摄像头。基于 CSI（信道状态信息）分析 WiFi 信号被人体散射的方式。

## 核心功能

- **人体姿态估计**：通过 WiFi 信号重建身体关键点位置
- **生命体征监测**：实时检测呼吸率和心率
- **穿墙存在检测**：穿透墙壁感知人类活动
- **自适应学习**：运行中学习本地环境，无需预标注数据

| 运动检测 | 5,400 倍提升 |
| 生命体征检测 | 11,665 帧/秒（单线程） |
| 完整管道 | 54,000 帧/秒 |

## 技术栈

- **纯 Rust 实现**，零外部 ML 依赖
- 信号处理：SpotFi 相位校正、Hampel 异常值排斥、菲涅尔区建模
- 56-192 个复数子载波值，采样频率 20Hz
- 物理学信号处理 + 机器学习结合

## 硬件部署

- 运行在 $9 ESP32 单片机上（~$1/节点网格部署）
- 完全离线，无需互联网
- 边缘 AI 模块直接运行在 ESP32 传感器上

## 为什么值得关注

将 WiFi 从网络通信工具转化为生物传感器，打开无摄像头人体感知新方向。基于 CMU 等机构的 WiFi DensePose 研究成果工程化，Rust 实现带来数百倍性能提升。隐私保护优于视频监控。
