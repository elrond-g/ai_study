# Vanna AI

> ⭐ 12k+ | Python | https://github.com/vanna-ai/vanna

## 项目简介

Vanna AI 是一个自然语言转 SQL 的开源工具，连接数据库后用户可以通过对话方式进行数据查询和分析，无需手写 SQL，极大降低了数据分析的技术门槛。

## 核心功能

- **自然语言转 SQL**：将用户的自然语言问题自动转换为准确的 SQL 查询
- **自动训练与优化**：基于数据库 DDL、文档和历史查询进行模型训练，提升转换准确率
- **可视化结果展示**：查询结果自动生成图表（Plotly），支持交互式数据探索

## 技术特点

- 支持 PostgreSQL、MySQL、SQLite、BigQuery、Snowflake 等多种数据库
- 采用 RAG 架构，利用向量检索匹配相似查询模式
- 可选多种 LLM 后端（OpenAI、Anthropic、本地模型等）

## 使用场景

- 业务人员自助式数据查询和分析
- 快速搭建企业数据问答机器人
- 数据库教学和 SQL 学习辅助工具
