# MathCode

> ⭐ 466 | https://github.com/math-ai-org/mathcode

## 项目简介

终端 AI 编程助手，专注数学形式化证明——将自然语言描述的数学问题自动转为 Lean 4 形式化定理并尝试自动证明。基于 AUTOLEAN 项目扩展，核心是"AI Agent 直接驱动 Lean Proof Assistant"。

## 核心特性

- **持久化 Lean REPL**：亚秒级编译检查反馈
- **定理/公理库**：证明过程可复用先前结果
- **Lean LSP 集成**：基于语言服务的引理发现
- **Obsidian 可视化**：定理依赖关系以知识图谱呈现
- **Agent 模式证明**：迭代式自动证明尝试
- **子目标树分解**：将复杂定理拆解为可管理的子目标
- **多 Planner 策略**：多种证明策略并行生成
- **可扩展**：通过 Skills / Tools / Plugins 扩展能力

## 技术栈

- Python + Shell
- Lean 4（定理证明器）
- 基于 AUTOLEAN 项目

## 使用场景

- 数学研究者形式化证明辅助
- 形式化数学教学（Lean 证明训练）
- 自动定理证明（ATP）研究
- AI 数学推理能力评估环境
