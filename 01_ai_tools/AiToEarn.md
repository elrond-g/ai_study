# AiToEarn

AI 驱动的全栈内容增长与变现平台，覆盖从创意生成、多渠道发布到数据分析与变现的完整链路。

## 核心特性

- **All In Agent**：AI Agent 自动完成内容创作和发布
- **14+ 平台分发**：抖音、快手、微信视频号、公众号、B站、小红书、Facebook、Instagram、TikTok、LinkedIn、Threads、YouTube、Pinterest、X/Twitter
- **AI 文案生成**：自动生成标题、描述和文案
- **AI 评论互动**：主动评论引流，提升曝光
- **AI 视频模型**：集成 Seedance、Kling、Hailuo、Veo、Sora、Pika、Runway 等
- **AI 图片模型**：支持 GPT、Flux
- **内容交易市场**：创作者直接售卖内容，品牌方采购优质素材
- **线下商业场景**（v1.8.0）：餐饮、零售、酒店、美容美发、健身房等实体商家线上推广

## 技术栈

- TypeScript 全栈
- Docker 一键部署（前端 + 后端 + MongoDB + Redis）
- MIT 开源协议
- v1.8.0 起聚焦 Web 版本
