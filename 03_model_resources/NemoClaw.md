# NemoClaw

> ⭐ N/A | Python | https://github.com/NVIDIA/NemoClaw

## 项目简介

NVIDIA 出品的 Agent 框架，用于构建可部署的对话式 AI Agent。基于 NVIDIA NeMo 生态，面向企业级 Agent 应用场景。

## 核心功能

- **对话式 Agent 构建**：提供完整的 Agent 开发工具链，支持多轮对话与工具调用
- **可部署架构**：内置生产级部署能力，适配 NVIDIA GPU 推理基础设施
- **NeMo 生态集成**：无缝对接 NeMo Framework 的模型训练与推理能力

## 技术特点

- 基于 NVIDIA 推理栈深度优化，充分利用 GPU 加速
- 支持自定义工具与外部 API 集成
- 提供标准化的 Agent 行为编排与状态管理

## 使用场景

- 企业内部智能客服与对话助手
- 基于 NVIDIA 基础设施的 Agent 部署
- 需要高性能推理的生产级 AI Agent 应用
