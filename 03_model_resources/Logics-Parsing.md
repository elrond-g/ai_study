# Logics-Parsing

阿里巴巴开源的端到端文档解析模型，基于 Qwen2.5-VL 视觉语言模型，采用强化学习优化复杂布局分析和阅读顺序推理。

## 核心特性

- **复杂文档处理**：报纸、杂志、学术论文等密集排版文档解析
- **多元素识别**：流程图（Mermaid）、乐谱（ABC 记谱法）、伪代码、化学式（SMILES）、科学公式
- **结构化 HTML 输出**：保留内容、元素类型、空间布局和语义层级
- **布局导向强化学习**：设计专用奖励机制优化布局分析和阅读顺序推理
- **高性能**：内部基准 82.16 分，OmniDocBench-v1.5 达 93.23 分

## 技术栈

- 基础模型：Qwen2.5-VL
- 训练方法：强化学习 + 设计奖励机制
- 输出格式：语义化 HTML
