# Spec Kit

> ⭐ N/A | TypeScript | https://github.com/github/spec-kit

## 项目简介

GitHub 官方出品的规范工具集，辅助 AI Agent 理解和遵守项目规范。让 AI 编码助手在生成代码时自动遵循团队约定。

## 核心功能

- **规范解析引擎**：自动解析项目中的代码规范与约定文件
- **Agent 集成接口**：为 AI Agent 提供标准化的规范查询能力
- **规范校验**：在 AI 生成代码时实时校验是否符合项目规范

## 技术特点

- 与 GitHub Copilot 生态深度集成
- 支持多种规范格式的自动识别与解析
- 轻量级设计，易于嵌入现有开发工作流

## 使用场景

- 让 AI 编码助手自动遵循团队代码风格
- 大型项目中统一 AI 生成代码的规范性
- CI/CD 流水线中校验 AI 产出是否合规
