# Open WebUI

> ⭐ 70k+ | TypeScript/Python | https://github.com/open-webui/open-webui

## 项目简介

自托管的 ChatGPT 风格 Web UI，支持 Ollama 和 OpenAI 等后端，是功能最丰富的本地 AI 对话前端之一。

## 核心功能

- **多模型对话界面**：支持同时接入多个 LLM 后端，自由切换模型对话
- **RAG 知识库**：内置文档上传与检索增强生成，支持 PDF/网页等多种格式
- **用户与权限管理**：支持多用户注册、角色权限控制，适合团队部署

## 技术特点

- 前端 SvelteKit + 后端 Python FastAPI 架构
- 支持插件系统与自定义工具扩展
- Docker 一键部署，开箱即用

## 使用场景

- 团队内部搭建私有 ChatGPT 替代方案
- 配合 Ollama 构建完整的本地 AI 对话系统
- 企业知识库问答与文档检索
