# Kreuzberg

> ⭐ 7,400+ | Rust | https://github.com/kreuzberg-dev/kreuzberg

## 项目简介

Kreuzberg 是一个多语言文档智能框架，Rust 核心实现，支持从 PDF/Office/图片等 91+ 格式提取文本、元数据、图片和结构化信息，覆盖 Rust/Python/Go/Java 等 12 种语言 SDK 及 CLI/REST/MCP 接口。

## 核心功能

- **91+ 格式支持**：PDF、Office、图片、电子书等全格式覆盖
- **多维提取**：文本、元数据、图片、结构化信息
- **12 种语言 SDK**：Rust/Python/Go/Java 等全栈支持
- **多接口**：CLI/REST API/MCP 三种访问方式
- **高性能**：Rust 核心实现，处理速度快

## 技术特点

- Rust 核心引擎
- 多语言 SDK 绑定
- MCP 协议原生支持

## 使用场景

- AI Agent 的文档内容提取预处理
- RAG 系统的文档解析层
- 批量文档格式转换和内容提取
