# Ollama

> ⭐ 120k+ | Go | https://github.com/ollama/ollama

## 项目简介

本地运行大语言模型的首选工具，一行命令即可下载并运行 Llama、Qwen、Gemma 等主流开源模型。极简设计，开箱即用。

## 核心功能

- **一键运行模型**：`ollama run llama3` 即可拉取并启动模型，体验极致简洁
- **模型库管理**：内置丰富的模型仓库，支持 GGUF 格式自定义导入
- **OpenAI 兼容 API**：提供与 OpenAI API 兼容的本地接口，方便已有项目无缝切换

## 技术特点

- 基于 llama.cpp 推理引擎，支持 CPU 和 GPU 混合推理
- 自动管理模型下载、量化版本选择与显存分配
- 跨平台支持 macOS / Linux / Windows

## 使用场景

- 本地开发调试 LLM 应用，避免 API 费用
- 隐私敏感场景下的离线模型推理
- 作为 Open WebUI 等前端的本地推理后端
