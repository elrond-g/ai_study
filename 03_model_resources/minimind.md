# MiniMind

开源的 LLM 从零训练框架，2 小时内从零训练 64M 参数的小型 GPT 模型。所有核心算法均基于原生 PyTorch 从头实现，极佳的 LLM 入门教学项目。

## 核心特性

- **快速训练**：2 小时从零训练 64M 参数 GPT
- **完整训练流水线**：预训练、SFT、LoRA、RLHF-DPO、RLAIF（PPO/GRPO/CISPO）、Tool Use、Agentic RL、自适应思维、模型蒸馏
- **纯 PyTorch 实现**：所有核心算法不依赖第三方框架封装，每行代码可读可理解
- **分布式训练**：支持 DDP 和 DeepSpeed 多 GPU 训练
- **视觉扩展**：MiniMind-V 项目 1 小时训练 26M 参数 VLM
- **长度外推**：YaRN 算法实现 RoPE 位置扩展

## 教学价值

专为降低 LLM 学习门槛设计，让每个人都能理解从分词到训练到评估的完整流程。
