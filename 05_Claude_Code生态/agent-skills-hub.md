# agent-skills-hub

> https://github.com/zhuyansen/agent-skills-hub

## 项目简介

agent-skills-hub 是一个社区驱动的 AI Agent Skills 聚合中心，收录和整理来自全球开发者贡献的 Claude Code Skills，提供分类索引和使用说明，是发现社区优质 Skills 的重要渠道。

## 核心内容

- **分类索引**：按功能领域组织的 Skills 目录
- **使用说明**：每个 Skill 的安装和使用指南
- **社区贡献**：开放提交，不断扩充新 Skills
- **质量评级**：基于社区反馈的 Skill 质量标注

## 收录的 Skill 类别

- 编程开发（前端/后端/全栈）
- 代码审查与重构
- 测试与 QA
- 文档写作
- 设计与 UI
- 数据分析
- DevOps 与运维
