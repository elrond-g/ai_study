# ui-design-brain

> https://github.com/carmahhawwari/ui-design-brain

## 项目简介

ui-design-brain 是一个超过 60 个 UI 组件最佳实践的 Claude Code Skill，内置 5 种主流设计风格的详细规范，让 AI 生成的 UI 代码真正符合专业设计标准。

## 五种设计风格

| 风格 | 特点 | 适用场景 |
|------|------|---------|
| SaaS | 清晰的层级、功能导向 | B2B 产品、管理后台 |
| 极简（Minimal） | 留白充足、克制配色 | 个人品牌、作品集 |
| 企业（Enterprise） | 专业严谨、信息密度高 | 企业软件、内部工具 |
| 创意（Creative） | 大胆配色、不规则布局 | 创意机构、活动页面 |
| 数据仪表盘 | 数据可读性优先 | 分析工具、监控面板 |

## 60+ 组件最佳实践

涵盖：
- **导航类**：Nav Bar、Sidebar、Breadcrumbs
- **表单类**：Input、Select、Checkbox、Date Picker
- **反馈类**：Toast、Modal、Alert、Loading
- **数据展示**：Table、Card、Chart、List
- **布局类**：Grid、Hero、Footer

## 使用场景

- 快速生成与产品定位匹配的 UI 组件
- 维护设计系统的一致性
- 开发者在无设计稿的情况下生成高质量 UI
