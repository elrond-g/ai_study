# impeccable / impeccable.style

> https://github.com/pbakaus/impeccable

## 项目简介

impeccable 是一套专注于前端代码质量和设计规范的 Claude Code Skill 集合，提供 17 个专业设计命令。作者 pbakaus 是 Google 的前 UX 工程师，将多年前端工程和设计系统经验编码为可复用的 AI Skills。

## 核心命令

| 命令 | 功能 |
|------|------|
| `/polish` | 全面提升 UI 细节质量 |
| `/audit` | 对现有代码做设计质量审查 |
| `/distill` | 提炼设计系统核心原则 |
| `/spacing` | 修复间距和节奏问题 |
| `/typography` | 优化字体层级和排版 |
| `/color` | 改善配色方案和对比度 |
| `/motion` | 添加流畅的动效 |

（共 17 个命令，覆盖 UI 质量提升全链路）

## 技术特点

- 每个 Skill 都内嵌了具体的设计原则和评判标准
- 命令可组合使用，按需叠加
- 与 Claude Code 深度集成，直接作用于当前项目代码

## 适合人群

- 全栈开发者希望提升 UI 代码质量但缺乏设计背景
- 设计系统维护者想要自动化规范检查
- 前端工程师建立个人设计质量标准
