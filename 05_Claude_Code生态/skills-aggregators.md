# Skills 聚合资源汇总

本文汇总 Claude Code / AI Agent Skills 的主要聚合平台，方便一站式发现优质 Skills。

## 主要聚合平台

### claude-skills
- GitHub: https://github.com/alirezarezvani/claude-skills
- 特点：由伊朗开发者维护，覆盖多语言场景，包含波斯语/英语双语 Skills

### ninehills/skills
- GitHub: https://github.com/ninehills/skills
- 特点：中文社区维护，包含专门针对中文使用场景的 Skills
- 亮点：`ai-avoid` Skill，让 AI 说话更像人，减少 AI 腔

### openai/skills
- GitHub: https://github.com/openai/skills
- 特点：OpenAI 官方维护，与 OpenAI 平台工具深度集成
- 适用：Codex、ChatGPT 生态用户

### awesome-openclaw-skills
- GitHub: https://github.com/VoltAgent/awesome-openclaw-skills
- 特点：OpenClaw（Claw）生态专属，与 Claw Agent 系统深度集成
- 覆盖：Claw 特有的工作流场景

### readyclaw.org
- 网址: https://www.readyclaw.org/en
- 特点：**数万个 Skills** 的在线平台，支持搜索和更新推送
- 最大规模的 Skills 聚合平台

## 选择建议

- **刚入门**：从 awesome-agent-skills 或 agent-skills-hub 开始，质量筛选好
- **数量优先**：readyclaw.org，最全面
- **中文场景**：ninehills/skills
- **Claw 用户**：awesome-openclaw-skills
