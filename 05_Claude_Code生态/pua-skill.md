# pua（Prompt 工程：让 AI 更努力工作）

> https://github.com/tanweai/pua

## 项目简介

pua 是一个基于大厂管理层思路的 Claude Code Skill，将企业管理中的激励和约束方法论迁移到 AI prompt 设计中——通过精心设计的语言策略让模型更认真、更彻底地完成任务。名字来源于中文互联网俚语"PUA"（精神控制/过度管理），带有一定的戏谑意味。

## 核心思想

普通 Prompt 和"PUA Prompt"的差异：
- **普通**：`帮我写一篇文章`
- **PUA 风格**：`这是你展示真正能力的机会，我需要你用最高标准完成...`

研究表明，在某些任务场景下，加入角色设定、紧迫感和高期望的描述确实能显著提升 LLM 输出质量。

## 核心功能

- **角色强化**：赋予 AI 高度专业化的角色定位
- **激励框架**：在 prompt 中加入明确的质量预期
- **约束设计**：通过特定约束引导 AI 更仔细地思考
- **大厂方法论**：将 OKR、KPI 等管理思维应用于 Prompt

## 使用场景

- 需要 AI 输出特别高质量结果的场景
- 学习 Prompt 心理学和激励机制设计
- 研究语言策略对 LLM 行为的影响
