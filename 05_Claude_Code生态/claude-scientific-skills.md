# claude-scientific-skills

> https://github.com/K-Dense-AI/claude-scientific-skills

## 项目简介

claude-scientific-skills 是 K-Dense AI 为生物科学研究场景开发的 Claude Code Skill 集合，将生物信息学、实验设计、数据分析等科研流程编码为 AI 可执行的标准化技能，加速生物科学研究的 AI 辅助工作流。

## 核心功能

- **实验设计辅助**：根据研究假设生成实验方案和对照组设计
- **生物信息分析**：基因组数据处理、序列比对、蛋白质结构分析
- **文献综述**：系统性检索和总结相关文献
- **统计分析**：选择合适的统计方法，解读实验结果
- **论文写作**：按生物科学期刊规范撰写实验报告

## 技术特点

- 内嵌生物科学领域知识和术语规范
- 符合 Nature/Science 系列期刊的写作风格要求
- 与生物信息学工具（BLAST、GATK 等）的使用指南集成

## 使用场景

- 生物科学研究者用 AI 加速实验设计和数据分析
- 研究生撰写实验报告和论文
- 跨学科团队中非生物背景成员快速上手生物数据分析
