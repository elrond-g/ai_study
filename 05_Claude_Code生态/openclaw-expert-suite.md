# openclaw-expert-suite

> https://github.com/rebootmindful/openclaw-expert-suite

## 项目简介

openclaw-expert-suite 是一个专家级多代理辩论系统，通过让多个具有不同视角的 AI "专家" 围绕同一问题进行辩论，推导出比单一 AI 回答更严密、更全面的结论。本质是一个复杂的 PUA 集中营——不同角色的压力促使 AI 输出更高质量的思考。

## 核心机制

**多专家辩论流程：**
1. 问题提交 → 分配给 N 个不同立场的专家 Agent
2. 各专家独立给出初始观点
3. 专家间交叉质疑和反驳
4. 综合者 Agent 汇总辩论结果，提炼最终结论

## 专家角色设计

- **提倡者**：为方案辩护，寻找支持证据
- **批评者**：挑战假设，找出漏洞
- **实用主义者**：从可执行性角度评估
- **风险分析师**：识别潜在问题和风险
- **综合者**：整合各方观点

## 使用场景

- 重要技术决策前的多维度评估
- 复杂问题的深度分析（投资决策、架构选型等）
- 学习如何设计多 Agent 辩论系统
