# llm-engineer-toolkit

> https://github.com/KalyanKS-NLP/llm-engineer-toolkit

## 项目简介

llm-engineer-toolkit 是由 KalyanKS-NLP 维护的 LLM 工程师学习资料聚合文档，收录 120+ 个 LLM 相关开发工具、库、课程和资源的入口，是 LLM 工程领域的综合性导航手册。

## 收录范围

### 开发框架
- **Orchestration**：LangChain、LlamaIndex、Haystack
- **Agent 框架**：AutoGen、CrewAI、LangGraph
- **Evaluation**：RAGAS、DeepEval、TruLens

### 模型工具
- **本地推理**：Ollama、llama.cpp、vLLM
- **微调**：Unsloth、Axolotl、TRL
- **量化**：GPTQ、AWQ、GGUF

### 可观测性
- LangSmith、Helicone、Langfuse、Phoenix

### 学习资源
- 斯坦福/CMU/MIT 的 LLM 相关课程
- 关键论文列表（Attention is All You Need 到最新研究）
- 实践教程和代码示例

## 为什么值得收藏

LLM 工程生态更新极快，新工具层出不穷。这份聚合文档帮助你：
- 快速了解某个方向的主流工具选型
- 发现可能遗漏的重要工具
- 跟踪 LLM 工程领域的技术全景
