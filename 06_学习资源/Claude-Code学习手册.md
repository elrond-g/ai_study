# Claude Code 学习手册（官方实战课程）

> Anthropic 官方 | https://anthropic.skilljar.com/claude-code-in-action

## 课程简介

这是 Anthropic 官方在 Skilljar 平台上发布的 Claude Code 实战课程，系统讲解如何高效使用 Claude Code 进行 AI 辅助软件开发。课程由 Anthropic 工程师主导，结合实际开发场景，是学习 Claude Code 最权威的第一手资料。

## 课程内容

**核心模块：**
1. **Claude Code 基础**：安装配置、基本对话和代码生成
2. **上下文管理**：如何用 CLAUDE.md 和 `.claudeignore` 优化上下文
3. **多文件项目**：处理大型代码库的策略
4. **工作流设计**：将 Claude Code 集成进开发流程
5. **高级技巧**：自定义 Skills、MCP 工具、成本优化

## 学习收益

- 理解 Claude Code 的工作原理和最佳实践
- 掌握不同开发场景下的使用策略
- 学会设计高效的 AI 辅助开发工作流
- 了解常见陷阱和避免方法

- 希望系统提升 Claude Code 使用效率的用户
