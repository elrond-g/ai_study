# everything-claude-code

> ⭐ 100,000+ | JavaScript | https://github.com/affaan-m/everything-claude-code | https://ecc.tools

## 项目简介

everything-claude-code 是一个 Agent Harness 性能优化系统，为 Claude Code、Codex、Opencode、Cursor 等 AI 编程工具提供 Skills、本能（Instincts）、记忆和安全层。核心目标是让 AI 编程 Agent 更快、更安全、更聪明。

## 核心功能

- **Skills 系统**：预置的专业能力集合，扩展 Agent 的工作范围
- **Instincts（本能）**：Agent 的默认行为规则，规避常见错误
- **Memory 系统**：跨会话记忆，累积项目知识
- **安全层**：防止危险操作，保护代码库
- **Research-first 开发模式**：先研究再实现的强制工作流

## 技术特点

- JavaScript 实现，轻量可嵌入
- 支持 Claude Code、Codex、Opencode、Cursor 多平台
- MCP 协议支持，可对接外部工具
- 完全开源（JavaScript），可直接阅读和修改

## 使用场景

- 企业为 AI 编程工具添加公司级别的安全约束
- 个人开发者系统化提升 AI 编程助手的能力上限
- 研究 AI Agent 的能力增强架构
