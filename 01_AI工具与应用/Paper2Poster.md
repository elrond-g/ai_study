# Paper2Poster

> ⭐ 3,501 | Python | https://github.com/Paper2Poster/Paper2Poster | https://paper2poster.github.io/

## 项目简介

Paper2Poster 是 NeurIPS 2025 D&B Track 入选论文，开源的多 Agent 学术海报自动生成系统。输入一篇 PDF 论文，系统自动理解核心贡献、提取关键图表，生成符合学术规范的 PPTX 格式学术海报。

## 核心功能

- **端到端生成**：从论文 PDF 到海报 PPTX，全自动流程
- **多 Agent 协作**：内容理解、布局设计、图表放置分由不同 Agent 负责
- **PPTX 输出**：生成可直接编辑的 PowerPoint 文件
- **关键贡献提取**：自动识别论文的核心方法、实验结果和主要发现

## 技术特点

- Python 实现，依赖 python-pptx 和多模态 LLM
- 支持对接 GPT-4V、Claude 等视觉语言模型
- 遵循学术海报排版规范（A0 尺寸、分栏布局等）

## 使用场景

- 研究人员快速将论文转化为会议展示海报
- 减少海报制作时间，专注于内容本身
- 批量生成多篇论文的展示材料
