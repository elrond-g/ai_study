# agency-agents

> ⭐ 60,000+ | Shell | https://github.com/msitarzewski/agency-agents

## 项目简介

agency-agents 是一个完整的 AI 代理团队集合，提供覆盖各类专业角色的 AI Agent 描述与工具链。项目的核心理念是：把一支真实的专业团队搬进 AI 工作流——前端工程师、Reddit 运营、内容审核、创意注入等角色各司其职，形成完整的"虚拟公司"。

## 核心功能

- **多角色 Agent 库**：涵盖前端开发、社区运营、质量把控、创意生成等数十个专业角色
- **角色人格设定**：每个 Agent 拥有独立的性格、工作流程和可交付成果定义
- **Shell 驱动**：以 Shell 脚本为核心，轻量无依赖，可快速集成到现有 CI/CD 流程
- **可扩展设计**：用户可基于已有角色模板自定义新角色

## 技术特点

- 使用 Shell 脚本定义角色指令，便于版本控制
- 角色描述结构化，包含职责（Role）、流程（Process）和交付物（Deliverables）三层
- 与 Claude Code、Codex 等 AI 编程工具深度集成

## 使用场景

- 构建多角色 AI 工作流，让不同 Agent 分工协作完成复杂任务
- 为 Claude Code 等工具提供预定义的专业 Skills
- 快速原型验证"0 人公司"或"AI-first 团队"工作模式
