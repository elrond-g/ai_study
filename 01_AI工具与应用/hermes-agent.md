# hermes-agent

> ⭐ 11,000+ | Python | https://github.com/NousResearch/hermes-agent | https://hermes-agent.nousresearch.com

## 项目简介

hermes-agent 是 Nous Research 出品的自学习 AI Agent 框架，核心理念是"与你一同成长的 Agent"。它不仅能执行任务，还能从用户的反馈和历史交互中不断积累经验，逐步提升自身能力，形成个性化的工作记忆。

## 核心功能

- **自学习机制**：从每次交互中提取经验，构建个人化知识库
- **多工具调用**：集成代码执行、网页搜索、文件操作等多种工具
- **长期记忆**：跨会话保存用户偏好和历史上下文
- **Claude/OpenAI 双栈**：同时支持 Anthropic 和 OpenAI 模型后端

## 技术特点

- 基于 Hermes 系列模型（Nous Research 自研），专为 Agent 任务优化
- 与 Claude Code、Codex、OpenClaw 等主流 AI 工具生态兼容
- 支持 MCP 协议，可对接外部工具服务

## 使用场景

- 构建会随使用越来越懂你的个人 AI 助手
- 研究 Agent 的持续学习和记忆架构
- 作为多模型 Agent 框架的基础层使用
