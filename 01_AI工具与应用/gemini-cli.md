# gemini-cli

> ⭐ 98,791 | TypeScript | https://github.com/google-gemini/gemini-cli | https://geminicli.com

## 项目简介

gemini-cli 是 Google 官方开源的 AI 命令行工具，将 Gemini 模型的能力直接带入终端。与 Claude Code 类似，它是一个完整的 AI 编程 Agent，支持代码生成、文件操作、命令执行和 MCP 工具调用。

## 核心功能

- **终端 AI Agent**：在命令行中与 Gemini 对话并执行任务
- **代码生成与修改**：理解项目上下文，直接修改代码文件
- **MCP 客户端**：支持 MCP 协议，可对接各类工具服务器
- **Google 服务集成**：原生集成 Google Search、Workspace 等服务
- **多模态支持**：可分析图片、截图等视觉内容

## 技术特点

- TypeScript 实现，与 Claude Code 架构相似
- 基于 Gemini 2.0/2.5 系列模型，支持 100 万 token 上下文
- 内置 Google Gemini API 免费额度，上手门槛低

## 与 Claude Code 对比

| 维度 | gemini-cli | Claude Code |
|------|------------|-------------|
| 背后模型 | Gemini 系列 | Claude 系列 |
| 上下文长度 | 100万 token | 20万 token |
| 免费额度 | 有（Gemini API） | 付费 |
| 开源程度 | 完全开源 | 部分开源 |
