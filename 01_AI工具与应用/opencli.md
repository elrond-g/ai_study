# opencli

> ⭐ 5,000+ | TypeScript | https://github.com/jackwener/opencli | https://opencli.info/

## 项目简介

opencli 是一个通用 CLI Hub 和 AI 原生运行时，能将任意网站、Electron 应用或本地二进制程序转化为标准化的命令行界面。通过 `AGENT.md` 集成规范，AI Agent 可以自动发现、学习并无缝调用各类工具。

## 核心功能

- **万物 CLI 化**：任意网站或工具均可生成对应的 CLI 接口
- **AGENT.md 规范**：标准化的工具描述文件，让 AI 自动理解工具用法
- **统一执行环境**：AI Agent 通过一个接口调用所有注册的工具
- **Electron 应用支持**：支持将桌面应用的功能暴露为 CLI 命令

## 技术特点

- TypeScript 实现，跨平台支持
- 基于 `AGENT.md` 的工具发现和自描述机制
- 与主流 AI Agent 框架（Claude Code、AutoGPT 等）兼容

## 使用场景

- 为 AI Agent 构建统一的工具调用层
- 将没有 API 的遗留系统接入 AI 工作流
- 快速为任意服务创建 AI 可用的命令行接口
