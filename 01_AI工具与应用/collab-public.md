# collab-public

> ⭐ 1,220 | TypeScript | https://github.com/collaborator-ai/collab-public

## 项目简介

Collaborator 是一个专为 AI Agent 协作设计的创作平台，让用户可以用 Agent 完成创意和工程工作。它重点解决多个 Claude（Claw）Agent 之间的工作流编排问题，允许用户可视化地设计 Agent 协作流程并执行。

## 核心功能

- **可视化工作流编排**：图形化设计 Agent 之间的任务传递关系
- **Claude Agent 协调**：统一管理多个 Claude 实例的执行
- **实时协作**：支持多用户同时参与 Agent 工作流的设计与调试
- **工作流持久化**：保存和复用已验证的 Agent 工作流模板

## 技术特点

- TypeScript + Next.js 前端
- 基于 Claude API 构建，支持 Claude 3/4 系列模型
- WebSocket 实现实时状态同步

## 使用场景

- 需要多个 AI Agent 分工合作完成复杂项目
- 构建可重复执行的 AI 自动化工作流
- 团队共同设计和调试 Agent 协作流程
