# superpowers

> ⭐ 107,000+ | Shell | https://github.com/obra/superpowers

## 项目简介

superpowers 是一个经过实践验证的 Agentic Skills 框架与软件开发方法论。它通过预定义的开发流程 Skills，让 AI Agent 遵循标准化的工程实践——自动脑暴、代码审查、测试驱动开发——而不是随意生成代码。

## 核心功能

- **Agentic Skills 框架**：提供一套可直接使用的 Claude Code Skills
- **自动脑暴（Auto Brainstorm）**：在实现前强制进行多角度方案探讨
- **结构化开发流程**：将软件工程最佳实践编码为 Agent 可执行的步骤
- **Shell 驱动**：轻量无依赖，与任何开发环境兼容

## 技术特点

- Shell 脚本实现，简单直接，易于理解和修改
- 每个 Skill 都有明确的输入、输出和工作流定义
- 支持与 Claude Code、Codex 等主流 AI 编程工具集成

## 使用场景

- 为 AI 编程 Agent 建立可重复、高质量的开发工作流
- 防止 AI 直接跳入实现而跳过设计思考
- 学习如何将软件工程实践转化为 AI Skills
