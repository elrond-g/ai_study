# skills-desktop

> ⭐ 242 | TypeScript | https://github.com/Harries/skills-desktop | https://skills.lc/desktop

## 项目简介

skills-desktop 是 skills.lc 平台的官方桌面客户端，提供图形化界面来管理和使用 Claude Code Skills。用户可以通过 GUI 浏览、安装、更新和组织 Skills，无需手动处理配置文件。

## 核心功能

- **Skills 可视化管理**：图形界面浏览和搜索 Skills 库
- **一键安装**：直接从 skills.lc 平台下载并激活 Skills
- **版本管理**：追踪已安装 Skills 的版本，支持更新
- **分类浏览**：按开发、设计、研究等类别筛选 Skills

## 技术特点

- TypeScript + Electron 桌面应用
- 与 skills.lc 云端 Skills 仓库同步
- 本地 Skills 配置文件自动管理

## 使用场景

- Claude Code 用户系统化管理自己使用的 Skills
- 快速发现和试用社区贡献的新 Skills
- 团队统一管理和分发标准化 Skills 配置
