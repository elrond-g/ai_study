### 成本优化

**代理层压缩**
- [rtk](https://github.com/rtk-ai/rtk)：做代理对 message 进行压缩，降低 Token 消耗

**配置层优化**

`.claudeignore` 忽略不必要的文件，示例：
```
node_modules/
dist/
build/
.next/
__pycache__/
*.lock
*.log
.git/
.idea/
.vscode/
coverage/
.cache/
```

`CLAUDE.md` 减少探索性读取（不写时 Claude Code 会大量 cat/find/grep 消耗 Token），示例：
```markdown
# 项目概述
Next.js 14 + TypeScript + Prisma + PostgreSQL 的 SaaS 应用

# 目录结构
src/
  app/          # App Router 页面
  components/   # React 组件
  lib/          # 工具函数
  server/       # 服务端逻辑

# 开发命令
pnpm dev        # 启动开发
pnpm build      # 构建
```

**任务描述优化**
- 任务要细化，描述直接：
  - ❌ `帮我重构整个用户模块`
  - ✅ `把 src/server/auth.ts 的 login 函数改成 async/await`
- `/compact` 压缩上下文，`/clear` 清理上下文
- 机械化工作让模型生成脚本/代码执行，不要每次依赖模型调度
