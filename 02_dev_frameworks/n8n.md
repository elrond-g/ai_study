# n8n

> ⭐ 50k+ | TypeScript | https://github.com/n8n-io/n8n

## 项目简介

开源工作流自动化平台，提供可视化编排界面，内置 400+ 集成节点，支持 AI Agent 节点和自托管部署。适合构建复杂的自动化流程和 AI 驱动的工作流。

## 核心功能

- **可视化工作流编辑器**：拖拽式界面构建自动化流程，零代码门槛
- **400+ 集成节点**：覆盖主流 SaaS、数据库、消息队列等服务的开箱即用连接器
- **AI Agent 节点**：原生支持 LLM 调用、Tool Use、RAG 等 AI 能力，可编排智能工作流

## 技术特点

- 支持自托管部署，数据完全掌控在自己手中
- 提供代码节点（JavaScript/Python），灵活处理自定义逻辑
- 支持 Webhook 触发、定时任务、事件驱动等多种启动方式

## 使用场景

- 企业内部流程自动化，如审批流、数据同步、通知推送
- 构建 AI Agent 工作流，串联 LLM 与外部工具完成复杂任务
- 监控数据源变化并自动触发下游动作
