# Excel MCP Server

> ⭐ N/A | Python | https://github.com/haris-musa/excel-mcp-server

## 项目简介

无需安装 Excel 即可操作表格文件的 MCP Server，让 AI Agent 直接读写 Excel 文件，支持数据分析、格式设置和公式操作。

## 核心功能

- **Excel 文件读写**：支持读取和写入 .xlsx 文件，包括多 Sheet 操作
- **数据分析处理**：支持筛选、排序、聚合等常见数据操作
- **格式与公式**：支持单元格格式设置和 Excel 公式的插入与计算

## 技术特点

- 基于 openpyxl 等 Python 库实现，无需安装 Microsoft Excel
- 遵循 MCP 协议标准，可被 Claude 等 AI 工具直接调用
- 轻量级部署，适合服务器端无 GUI 环境运行

## 使用场景

- 让 AI Agent 自动生成和填充 Excel 报表
- 批量处理 Excel 数据，如清洗、转换、合并多个文件
- 在无 Excel 环境的服务器上进行表格数据的自动化处理
