# Task Master AI

> ⭐ N/A | JavaScript | https://github.com/eyaltoledano/claude-task-master

## 项目简介

AI 项目经理 MCP Server，将 PRD 自动分解为结构化任务并逐个执行，为 AI 驱动的开发流程提供任务管理和编排能力。

## 核心功能

- **PRD 任务分解**：输入产品需求文档，自动拆解为可执行的结构化子任务
- **任务依赖管理**：识别任务间的依赖关系，按正确顺序编排执行
- **进度追踪**：跟踪每个任务的完成状态，支持任务的迭代和调整

## 技术特点

- 基于 MCP 协议，可与 Claude Code、Cursor 等 AI 开发工具集成
- 任务以 JSON 格式持久化存储，支持跨会话恢复
- 支持任务粒度的自动调整，过大的任务会被进一步拆分

## 使用场景

- AI 辅助开发时将需求文档转化为可执行的开发计划
- 管理复杂项目的任务分解和执行顺序
- 团队协作中用 AI 自动生成和分配开发任务
