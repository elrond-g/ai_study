# Langflow

> ⭐ N/A | Python | https://github.com/langflow-ai/langflow

## 项目简介

基于 LangChain 生态的可视化 LLM 应用构建工具，通过拖拽方式编排 Agent、RAG、Chain 等组件，快速搭建和迭代 AI 应用原型。

## 核心功能

- **可视化编排**：拖拽式画布构建 LLM 应用，实时预览运行结果
- **丰富组件库**：内置 Agent、Chain、Retriever、Memory 等 LangChain 核心组件
- **一键导出**：将可视化流程导出为 Python 代码或 API 端点，方便集成到生产环境

## 技术特点

- 深度集成 LangChain 生态，与其组件无缝衔接
- 支持自定义组件开发，可扩展私有节点
- 提供 REST API 接口，构建好的流程可直接作为服务调用

## 使用场景

- 快速原型验证 RAG、Agent 等 LLM 应用方案
- 非开发人员通过可视化方式构建 AI 工作流
- 团队协作调试和优化 LangChain 应用链路
