# Huginn

> ⭐ 46k+ | Ruby | https://github.com/huginn/huginn

## 项目简介

自托管的自动化 Agent 系统，能够监控网页变化、API 数据并自动触发动作，类似开源版 IFTTT/Zapier，适合构建个人自动化信息管道。

## 核心功能

- **多源数据监控**：监控网页内容变化、RSS 订阅、API 返回值等多种数据源
- **事件驱动 Agent**：多个 Agent 协作，事件在 Agent 之间流转形成自动化链路
- **灵活的动作触发**：支持发送邮件、Webhook 回调、生成 RSS、执行脚本等输出动作

## 技术特点

- 完全自托管，所有数据存储在本地，保护隐私
- Agent 之间通过事件传递机制解耦，易于组合和扩展
- 提供 Web 管理界面，可视化查看 Agent 状态和事件流

## 使用场景

- 监控竞品网站价格变动或内容更新并自动通知
- 聚合多个信息源生成个性化 RSS 或摘要推送
- 定时抓取 API 数据并写入数据库或触发下游流程
